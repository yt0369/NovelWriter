# 核心指令：历史架空角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 历史架空示例 |
|-----|------|-------------|
| 出身 | 背景、起点 | 寒门士子、世家子弟、皇族宗室、异族质子、流民首领 |
| 核心经历 | 塑造性格的关键事件 | 战乱流离、政变失败、家族被灭、科举落第 |
| 创伤/执念 | 促成行为模式的心理锚点 | "恢复家族荣耀"、"改变历史走向"、"夺回皇位" |

**底色公式**：
```
底色 = 历史起点 + 核心经历 → 形成的政治态度
```

**历史架空示例**：
- 主角底色：寒门士子 + 科举舞弊落第 → "痛恨腐败、立志改革"
- 对手底色：世家嫡子 + 从小接受精英教育 → "维护门阀利益、保守传统"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 权力型 | 夺取皇位、掌握朝政 | 登基称帝、权倾朝野 |
| 改革型 | 变法图强、富国强兵 | 改革成功、国力提升 |
| 复仇型 | 恢复家族、报仇雪恨 | 复仇成功、家族复兴 |
| 理想型 | 治国平天下、造福百姓 | 盛世来临、百姓安居 |
| 军功型 | 建功立业、封侯拜将 | 战功赫赫、封王封侯 |

**追求的三个层次**：
```
表层追求：具体目标（平定叛乱、通过科举、击败某个政敌）
深层追求：身份认同（证明自己的能力、被认可为贤臣/明君）
终极追求：存在意义（改变历史、留名青史、开创盛世）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 政治智慧 | 权谋、联盟、制衡 | 纵横捭阖、合纵连横、制衡派系 |
| 军事才能 | 战术、统兵、后勤 | 用兵如神、练兵有方、粮草充足 |
| 历史知识 | 预知历史走向 | 提前布局、避免灾难、把握机遇 |
| 人格魅力 | 收服人心 | 礼贤下士、知人善任、恩威并施 |
| 经济手段 | 理财、商业、税收 | 开源节流、发展商业、充实国库 |

**历史架空手段设计原则**：
1. **有短板**：不能所有方面都最强（政治强但军事弱）
2. **有成长**：能力要能提升（从书生到名将的成长线）
3. **有特色**：要有标志性的手段（独特的改革方案、战术）
4. **有对手**：对手也要有特色和实力

---

## 二、历史架空爽点定位设计

> 历史架空角色的核心价值：**读者看他权谋、打仗、改革爽不爽？**

### 爽点类型矩阵（历史架空特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **政治胜利** | 权谋斗争中获胜 | 解气、佩服 | 权谋大师、政治家 |
| **军事胜利** | 战争中击败敌人 | 热血、振奋 | 名将、军事天才 |
| **改革成功** | 变法图强成功 | 励志、感动 | 改革家、理想主义者 |
| **收服人才** | 招揽名士名将 | 满足、期待 | 明君、伯乐 |
| **改变历史** | 扭转历史走向 | 震撼、兴奋 | 穿越者、历史改变者 |
| **打脸权贵** | 击败世家门阀 | 解气、痛快 | 寒门逆袭、平民英雄 |

```

- 主爽点：政治胜利 + 军事胜利
- 辅爽点：改革成功
- 爽点触发条件：朝堂斗争、战场厮杀、改革推进
- 爽点表现形式：权谋击败政敌 + 战场击败敌军

【历史设定】
- 朝代背景：[汉/唐/宋/明/架空]
- 主角身份：
- 历史节点：
- 架空程度：
- 主要矛盾：

【成长路线】
- 短期目标：科举及第/平定叛乱
- 中期目标：入阁拜相/封侯拜将
- 长期目标：改变历史/开创盛世
```

# 核心指令：历史架空期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化历史剧情的爽点节奏
*   读者反馈"不够紧张"或"权谋不精彩"
*   规划重要事件前的铺垫
*   设计主角成长曲线

---

## GAP 三角模型：历史架空期待感引擎

```
【差距 Gap】→ 实力差距/地位差距（读者："能赢吗？"）
     ↓
【期待 Anticipation】→ 布局谋划/练兵备战（读者："有戏！准备好了！"）
     ↓
【满足 Payoff】→ 政治胜利/战争胜利（读者："爽！赢了！太精彩了！"）
```

### 历史架空期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 权谋期待 | 即将朝堂斗争 | 政治胜利 | 紧张→解气 | 击败政敌、扳倒权臣 |
| 军事期待 | 即将战争 | 战争胜利 | 紧张→振奋 | 大败敌军、收复失地 |
| 改革期待 | 推行变法 | 改革成功 | 期待→励志 | 变法成功、国力提升 |
| 历史期待 | 历史转折点 | 扭转历史 | 期待→震撼 | 改变历史走向 |
| 收服期待 | 招揽人才 | 人才归附 | 期待→满足 | 名将归附、贤士投效 |

---

## 满足感分级详解（历史架空）

## 权谋节奏设计

### 单次权谋节奏

```
铺垫期：矛盾积累、派系对立
布局期：暗中谋划、联络盟友
交锋期：朝堂对峙、明争暗斗
高潮期：关键时刻、一击制胜
收尾期：政敌倒台、权力重组
```

### 战争节奏设计

```
备战期：练兵、筹粮、制定战略
开战期：初次交锋、试探虚实
拉锯期：反复争夺、互有胜负
决战期：决定性战役、一战定乾坤
收尾期：论功行赏、巩固成果
```

## 历史架空爽点三级分类

### 小爽点（+1~+2）
**定义**：小规模胜利、收服人才

**典型场景**：
- 小规模战斗胜利
- 政治小胜（击败小政敌）
- 收服小人物（谋士、将领）
- 科举及第
- 获得皇帝赏识
- 改革小成（某项政策成功）
- 打脸看不起自己的人
- 发现历史机遇

**频率建议**：每1章一次

**作用**：维持进展感，让读者感受到持续成长

---

### 中爽点（+3~+4）
**定义**：重要战役胜利、扳倒权臣

**典型场景**：
- 重要战役胜利
- 扳倒权臣/政敌
- 改革推进（重要改革成功）
- 收服名将/名士
- 晋升高位（入阁拜相）
- 平定叛乱
- 收复失地
- 获得兵权/财权

**频率建议**：每5章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：统一天下、变法成功

**典型场景**：
- 统一天下
- 变法成功（国力大增）
- 权倾朝野
- 登基称帝
- 改变历史走向
- 开创盛世
- 击败宿敌（灭国/灭族）

**频率建议**：每20章一次

**作用**：全书记忆点，读者口碑传播点

## 爽点节奏配置（历史架空推荐）

### 高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 每章都有进展/胜利 |
| 中爽间隔 | 5章 | 频繁的重要胜利 |
| 大爽间隔 | 20章 | 重大历史事件 |

### 配置原则
1. **进展不断**：最长不超过2章无权谋/军事进展
2. **对手递进**：对手实力要逐步提升
3. **权力成长**：权力要有可见的提升
4. **历史推进**：历史进程要有明显变化

# 核心指令：历史架空世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要构建历史世界的基础设定
*   需要设计政治格局或军事体系
*   需要规划经济制度或社会结构
*   世界观设定出现矛盾或漏洞

---

## 一、历史背景核心三要素

### 1. 朝代设定

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 朝代选择 | 基于哪个朝代 | 汉/唐/宋/明/架空 |
| 历史节点 | 处于什么时期 | 开国/盛世/衰落/乱世 |
| 架空程度 | 与真实历史的偏离 | 完全架空/半架空/平行世界 |

**朝代设定原则**：
1. **历史感**：要有真实历史的厚重感
2. **合理性**：架空要有合理的历史逻辑
3. **冲突性**：要有足够的矛盾和冲突
4. **发展性**：要有改变和发展的空间

### 2. 政治格局

**中央朝廷**：
```
皇帝
  ├─ 内阁/中书省（行政）
  ├─ 六部（吏/户/礼/兵/刑/工）
  ├─ 都察院（监察）
  ├─ 翰林院（文化）
  └─ 宦官/外戚（权力寄生）
```

**地方势力**：
- 藩王：宗室势力
- 节度使/总督：地方军政
- 世家门阀：地方豪强
- 异族势力：边疆威胁

**派系划分**：
- 皇权派 vs 相权派
- 改革派 vs 保守派
- 文官集团 vs 武将集团
- 寒门士子 vs 世家门阀

### 3. 社会结构

| 阶层 | 地位 | 权力 | 矛盾 |
|-----|------|------|------|
| 皇族 | 最高 | 皇权 | 皇位争夺 |
| 世家门阀 | 极高 | 政治/经济 | 与皇权/寒门的矛盾 |
| 士大夫 | 高 | 文化/政治 | 派系斗争 |
| 商人 | 中 | 经济 | 重农抑商 |
| 平民 | 低 | 无 | 生存压力 |
| 奴隶/贱民 | 最低 | 无 | 阶层固化 |

## 二、军事体系

### 兵种分类

| 兵种 | 特点 | 作用 | 克制关系 |
|-----|------|------|---------|
| 步兵 | 数量多、成本低 | 主力 | 被骑兵克制 |
| 骑兵 | 机动性强 | 突击 | 克制步兵、被弓兵克制 |
| 弓兵 | 远程攻击 | 压制 | 克制骑兵、被步兵克制 |
| 水军 | 水上作战 | 江河湖海 | 地形限制 |
| 攻城器械 | 攻城 | 破城 | 移动缓慢 |

### 军事制度

**募兵制 vs 征兵制**：
- 募兵制：职业军人，战力强，成本高
- 征兵制：临时征召，战力弱，成本低

**军队组织**：
```
大将军/元帅
  ├─ 前军（先锋）
  ├─ 中军（主力）
  ├─ 后军（殿后）
  ├─ 左军（侧翼）
  └─ 右军（侧翼）
```

### 战术体系

**常见战术**：
- 正面强攻：以力破力
- 侧翼包抄：迂回包围
- 诱敌深入：设伏歼灭
- 声东击西：调虎离山
- 坚壁清野：消耗敌军
- 奇袭夜袭：出其不意

## 三、经济制度

### 财政来源

| 来源 | 说明 | 占比 |
|-----|------|------|
| 田赋 | 农业税 | 60% |
| 商税 | 商业税 | 20% |
| 盐铁专卖 | 国家垄断 | 15% |
| 其他 | 矿税、关税等 | 5% |

### 货币体系

**货币类型**：
- 铜钱：日常交易
- 银两：大额交易
- 金锭：储备货币
- 纸币：（宋以后）

**汇率关系**：
```
1两金 = 10两银 = 10贯钱 = 10000文
```

### 土地制度

**土地分配**：
- 均田制：国家分配
- 庄园制：地主占有
- 租佃制：佃农耕种

**土地兼并**：
- 世家门阀大量占地
- 农民失地成为流民
- 社会矛盾激化

## 四、科举制度

### 科举等级

| 等级 | 考试 | 地位 | 出路 |
|-----|------|------|------|
| 童生 | 童试 | 最低 | 继续考 |
| 秀才 | 院试 | 低 | 免徭役 |
| 举人 | 乡试 | 中 | 可为官 |
| 进士 | 会试+殿试 | 高 | 入仕 |
| 状元/榜眼/探花 | 殿试前三 | 极高 | 高官 |

### 科举内容

**考试科目**：
- 经义：四书五经
- 策论：时政对策
- 诗赋：文学才华
