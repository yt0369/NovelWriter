# 核心指令：网游副本流角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 网游副本示例 |
|-----|------|-------------|
| 出身 | 背景、起点 | 平民玩家、公会精英、散人高手、氪金大佬、内测玩家 |
| 核心经历 | 塑造性格的关键事件 | 公会背叛、副本团灭、装备被骗、首杀失败 |
| 创伤/执念 | 促成行为模式的心理锚点 | "成为服务器第一"、"打通最难副本"、"复仇背叛者" |

**底色公式**：
```
底色 = 网游起点 + 核心经历 → 形成的游戏态度
```

**网游副本示例**：
- 主角底色：被公会背叛的首杀团长 + 装备被骗退游两年 → "谨慎低调、用实力说话"
- 对手底色：服务器第一公会会长 + 从未失败过 → "骄傲自负、渴望真正的对手"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 首杀型 | 追求副本首杀、世界第一 | 服务器首杀、极限通关 |
| 收集型 | 追求极品装备、稀有坐骑 | 神装掉落、收集成就 |
| 公会型 | 建立/带领公会称霸服务器 | 公会战胜利、占领城池 |
| 探索型 | 发现隐藏任务、探索未知地图 | 发现秘密、隐藏奖励 |
| 复仇型 | 击败曾经背叛/击败自己的人 | 复仇成功、夺回装备 |

**追求的三个层次**：
```
表层追求：具体目标（通关某个副本、获得某件装备）
深层追求：身份认同（证明自己是最强的、被认可为大神）
终极追求：存在意义（成为传奇、留下自己的印记）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 操作技术 | 走位、技能衔接、反应 | 完美走位、技能连招、极限躲避 |
| 副本经验 | 机制理解、Boss 打法 | 熟悉机制、团队配合、指挥能力 |
| 装备优势 | 极品装备、套装效果 | 神装、套装、属性加成 |
| 职业特色 | 职业技能、天赋加点 | 本命职业、流派选择、技能搭配 |
| 资源获取 | 金币、材料、人脉 | 商业头脑、情报网络、公会资源 |

**网游副本手段设计原则**：
1. **有短板**：不能所有方面都最强（操作强但装备差）
2. **有成长**：能力要能提升（从萌新到大神的成长线）
3. **有特色**：要有标志性的打法（招牌职业、独特战术）
4. **有对手**：对手也要有特色和实力

## 二、网游副本爽点定位设计

> 网游副本角色的核心价值：**读者看他打副本、拿装备、PK 爽不爽？**

### 爽点类型矩阵（网游副本特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **副本通关** | 首杀、极限通关、单刷 | 震撼、佩服 | 副本之王、首杀狂魔 |
| **装备获得** | 极品掉落、毕业装备 | 惊喜→满足 | 欧皇、神装收集者 |
| **PK 碾压** | 野外 PK、公会战 | 解气、痛快 | PK 狂人、公会战神 |
| **成长蜕变** | 从萌新到大神 | 励志、感动 | 努力型、逆袭型 |
| **智慧碾压** | 用战术智慧击败对手 | 佩服、过瘾 | 战术大师、指挥官 |
| **探索发现** | 发现隐藏任务、秘密 | 惊喜、兴奋 | 探索者、秘密猎人 |


# 核心指令：网游副本期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化网游剧情的爽点节奏
*   读者反馈"不够刺激"或"副本不紧张"
*   规划重要副本前的铺垫
*   设计主角成长曲线

---

## GAP 三角模型：网游副本期待感引擎

```
【差距 Gap】→ 副本难度/装备差距（读者："能通关吗？"）
     ↓
【期待 Anticipation】→ 装备提升/战术准备（读者："有戏！准备好了！"）
     ↓
【满足 Payoff】→ 副本通关/装备掉落（读者："爽！通关了！神装到手！"）
```

### 网游副本期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 副本期待 | 即将挑战高难副本 | 副本通关/首杀 | 紧张→兴奋 | 挑战服务器最难副本 |
| 成长期待 | 等级/装备瓶颈 | 等级突破/装备升级 | 期待→满足 | 满级、毕业装备 |
| 掉落期待 | Boss 即将击杀 | 极品装备掉落 | 期待→惊喜 | 打出神装、稀有坐骑 |
| 公会期待 | 公会战/领地战 | 公会胜利 | 热血→自豪 | 占领城池、击败敌对公会 |
| 复仇期待 | 曾经被击败/背叛 | 复仇成功 | 解气→痛快 | 击败仇人、夺回装备 |

---

## 满足感分级详解（网游副本）

| 等级 | 情绪值 | 触发频率 | 典型形式 | 网游示例 |
|-----|-------|---------|---------|---------|

| 大爽 | +4 | 全书2-3次 | 重大转折 | 服务器首杀、神装获得、公会称霸 |
| 极爽 | +5 | 全书1次 | 终极高潮 | 打通终极副本、成为传奇 |

## 副本节奏设计

### 单个副本节奏

```
准备期：组队、装备检查、战术讨论
开荒期：摸索机制、多次失败、逐步进步
突破期：掌握机制、配合提升、接近成功
高潮期：极限操作、惊险通关、首杀达成
收获期：装备分配、成就获得、庆祝
```

### 版本周期节奏

```
新版本开启（小爽：新地图探索）→ 普通副本（小爽：装备积累）
→ 困难副本（中爽：首杀成功）→ 公会发展（小爽：招募成员）
→ 公会战（中爽：击败敌对公会）→ 史诗副本（中爽：服务器首杀）
→ 终极副本（大爽：神装获得）
```

# 核心指令：网游副本爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 网游副本爽点三级分类

### 小爽点（+1~+2）
**定义**：日常进展、小胜利

**典型场景**：
- 小怪掉落蓝装/紫装
- 完成日常任务
- 野外 PK 击杀敌对玩家
- 发现隐藏任务
- 获得队友认可
- 普通副本通关
- 技能连招成功
- 等级提升

**频率建议**：每1章一次

**作用**：维持进展感，让读者感受到持续成长

---

### 中爽点（+3~+4）
**定义**：重要副本通关、极品装备

**典型场景**：
- 困难副本首杀
- Boss 掉落极品装备
- 等级突破（满级）
- 公会战胜利
- 击败强力对手
- 获得稀有坐骑/宠物
- 完成史诗任务链
- 套装凑齐

**频率建议**：每5章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：服务器首杀、神装获得

**典型场景**：
- 服务器首杀最难副本
- 获得神器/传说装备
- 公会占领主城
- 击败服务器第一公会
- 打通终极副本
- 成为服务器传奇
- 创造游戏历史记录

**频率建议**：每20章一次

**作用**：全书记忆点，读者口碑传播点

---

## 爽点节奏配置（网游副本推荐）

### 高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 每章都有进展/掉落 |
| 中爽间隔 | 5章 | 频繁的副本通关/装备获得 |
| 大爽间隔 | 20章 | 重大成就达成 |

### 配置原则
1. **进展不断**：最长不超过2章无副本/装备进展
2. **副本递进**：副本难度要逐步提升
3. **装备成长**：装备要有可见的提升
4. **对手升级**：对手实力要逐步增强

# 核心指令：网游副本世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要构建网游世界的游戏系统设定
*   需要设计副本机制和Boss战
*   需要规划装备体系和成长路线
*   游戏设定出现数值失衡或逻辑漏洞

---

## 一、网游世界观核心三要素

### 1. 游戏系统
游戏运行的基础规则。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 游戏类型 | VRMMO/键盘网游/全息 | 影响玩家的交互方式 |
| 核心玩法 | 副本攻略/PVP/收集/养成 | 决定主要游戏内容 |
| 职业体系 | 战法牧等职业 | 6-8个基础职业，有转职 |
| 等级上限 | 当前版本最高等级 | 分阶段开放（如60→80→100） |
| 死亡惩罚 | 角色死亡后的后果 | 掉经验/掉装备/无惩罚 |

**游戏类型选择**：
```
VRMMO（全息虚拟现实）
├── 特点：沉浸感强、身体操作
├── 优势：战斗描写更精彩
└── 适合：副本攻略流

键盘网游
├── 特点：技能释放、数据化操作
├── 优势：策略性更强
└── 适合：竞技/公会战流

混合型
├── 特点：半全息+数据面板
├── 优势：兼顾沉浸和策略
└── 适合：综合型网游文
```

### 2. 职业体系

| 类型 | 职业 | 定位 | 团队角色 |
|-----|------|------|---------|
| 坦克 | 骑士/守护者 | 承伤 | 吸引Boss仇恨 |
| 输出(近战) | 战士/刺客 | 爆发伤害 | 切后排/持续输出 |
| 输出(远程) | 法师/弓手 | 范围伤害 | AoE/远程输出 |
| 治疗 | 牧师/药师 | 回血增益 | 维持团队生存 |
| 辅助 | 吟游诗人/召唤师 | 控制/增益 | 团队增益 |
| 隐藏职业 | 特殊职业 | 特殊能力 | 独一无二的玩法 |

### 3. 副本系统

**副本分级**：
```
普通副本（5人）→ 精英副本（5人困难）→ 团队副本（10人）
    → 大型团队副本（25人）→ 史诗副本（40人）→ 传奇副本（全服唯一）
```

**副本设计模板**：
```
【副本名称】
- 难度等级：普通/精英/英雄/史诗/传奇
- 推荐等级：Lv.XX-Lv.XX
- 人数限制：X人
- 副本时长：XX分钟
- Boss数量：X个

【Boss设计】
Boss 1：[小Boss] - 测试基础配合
Boss 2：[中Boss] - 需要特殊机制
Boss 3：[最终Boss] - 多阶段战斗

【掉落表】
- 普通掉落：装备/材料/金币
- 稀有掉落：蓝色装备/配方
- 史诗掉落：紫色装备/技能书
- 传说掉落：橙色装备/坐骑
```

## 二、装备与经济体系

### 装备品质

| 品质 | 颜色 | 获取方式 | 属性倍率 |
|-----|------|---------|---------|
| 普通（白） | 白色 | 小怪掉落 | 基础 |
| 优秀（绿） | 绿色 | 精英怪掉落 | 1.2x |
| 精良（蓝） | 蓝色 | 副本Boss掉落 | 1.5x |
| 史诗（紫） | 紫色 | 困难副本/合成 | 2x |
| 传说（橙） | 橙色 | 史诗副本/任务 | 3x |
| 神话（红） | 红色 | 传奇副本/世界事件 | 5x |

### 经济系统

```
【货币体系】
铜币 → 基础货币（日常消耗）
银币 → 中级货币（装备交易）
金币 → 高级货币（稀有物品）
钻石 → 充值货币（商城道具）

【经济循环】
打怪/副本产出 → 交易/拍卖 → 装备升级 → 更强战力 → 更高副本
```

## 四、世界观设计检查清单

- [ ] 游戏系统规则是否自洽？（有没有逻辑漏洞）
- [ ] 职业体系是否平衡？（每个职业都有用）
- [ ] 副本设计是否有挑战性？（不能太简单也不能太难）
- [ ] 装备体系是否有成长感？（品质递进清晰）
- [ ] 经济系统是否平衡？（避免通货膨胀）
- [ ] 公会系统是否有竞争性？（公会战、领地争夺）
- [ ] 数值体系是否合理？（避免战力崩坏）
- [ ] 世界观是否为副本攻略服务（而非为设定而设定）？
