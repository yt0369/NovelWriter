# 核心指令：克苏鲁/宇宙恐怖角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑，或理智崩溃缺乏铺垫

---

## 一、核心模型：底色-追求-代价三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 宇宙恐怖示例 |
|-----|------|-------------|
| 出身 | 职业、社会身份 | 民间传说学者、精神科医生、灯塔看守人、退休警探 |
| 接触前状态 | 接触异常之前的正常生活 | 平静的学术生涯、温馨的家庭生活、安稳的退休时光 |
| 初始理智 | 心理韧性与精神承受力 | 极高（军人/刑警）、中等（普通学者）、偏低（敏感体质者） |
| 暗线伏笔 | 角色不自知的与异常的联系 | 家族诅咒、童年被抹去的记忆、反复出现的梦境 |

**底色公式**：
```
底色 = 接触异常前的平静生活 + 隐含的异常关联 → 形成的对未知的态度
```

**宇宙恐怖示例**：
- 主角底色：冷静理性的考古学教授 + 家族祖传的不可解读的手稿 → "用理性解释一切"直到理性崩塌
- 反派底色：天才科学家 + 偶然接触古抄本 → "知识即力量"走向"知识即疯狂"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 求生型 | 在异常事件中活下去 | 死里逃生、规则破解、侥幸存活 |
| 求知型 | 追求禁忌知识的真相 | 发现线索、拼凑真相、理解规则 |
| 守护型 | 保护他人免受恐怖侵蚀 | 以代价换取他人安全 |
| 救赎型 | 弥补自己犯下的错误 | 阻止灾难、封印实体、逆转仪式 |
| 对抗型 | 主动对抗不可名状之物 | 人类意志的悲壮抵抗 |

**追求的三个层次**：
```
表层追求：具体目标（活着离开这个地方/查明失踪案的真相）
深层追求：认知定位（理解这个世界到底怎么了/我是不是已经疯了）
终极追求：存在意义（人类是否有资格了解真相/知道真相后还能保持人性吗）
```

---

### 3. 代价（Cost）── 他要付出什么？

| 代价类型 | 说明 | 典型形式 |
|---------|------|---------|
| 理智损耗 | 每次接触真相都在侵蚀心智 | 幻觉、偏执、失眠、人格分裂 |
| 记忆代价 | 真相的获取伴随记忆的丧失 | 忘记重要的人、时间感错乱、身份认知模糊 |
| 肉体代价 | 接触异常的生理影响 | 异变征兆、慢性疼痛、感官异化 |
| 关系代价 | 身边的人因自己而遭受牵连 | 亲友失踪/异变、被社会孤立 |
| 灵魂代价 | 终极代价——自我的本质被改变 | 失去"人"的属性、成为容器的风险 |

**宇宙恐怖代价设计原则**：
1. **对等交换**：每一个重要发现必须对应等价的代价
2. **不可逆转**：付出的代价无法完全恢复（伤疤永远存在）
3. **渐进侵蚀**：代价是缓慢累积的，角色一开始不自知
4. **债务倍率**：1.3（读者预期每次收获都附带隐藏成本）

## 二、克苏鲁角色爽点定位设计

> 宇宙恐怖角色的核心价值：**读者看他面对未知时的挣扎、推理、抉择——既害怕真相又渴望真相。**

### 爽点类型矩阵（宇宙恐怖特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **规则破解** | 从混乱中梳理出生存规则 | 智识上的满足 | 冷静分析者、规则猎手 |
| **死里逃生** | 以极限操作逃出必死之局 | 心跳加速、庆幸 | 求生专家、直觉敏锐者 |
| **真相拼图** | 发现关键线索，拼凑出真相 | 恍然大悟、背后发凉 | 学者型侦探、禁书收藏家 |
| **意志抵抗** | 以凡人之躯对抗不可名状 | 悲壮、敬畏 | 绝不屈服者、牺牲者 |
| **禁忌诱惑** | 在求知欲与自保间挣扎 | 紧张、矛盾 | 禁忌知识追寻者 |
| **微光希望** | 在绝望中找到一丝温暖 | 感动、珍惜 | 守护者、暗夜中的火炬 |

### 主角爽点定位模板（宇宙恐怖）

```
【爽点定位】
- 主爽点：规则破解 + 真相拼图
- 辅爽点：死里逃生
- 爽点触发条件：面对未知异常、发现关键线索时
- 爽点表现形式：冷静分析中突然顿悟，在绝望中找到生路

【理智系统】
- 初始理智值：80/100
- 理智损耗来源：目睹异常、阅读禁忌典籍、接触实体
- 理智恢复方式：安全区域休息、人际联结、日常仪式感
- 理智崩溃阈值：30（开始出现幻觉）、15（行为失控）、0（永久疯狂）

【知识代价路线】
- 短期代价：噩梦、短暂幻觉
- 中期代价：记忆缺失、感官异化、关系疏离
- 长期代价：人格裂痕、肉体异变征兆、存在感稀薄
```

## 五、检查清单

### 主角必查项
- [ ] 底色是否足够"正常"？（正常人才有反差恐惧）
- [ ] 理智系统是否有明确数值和规则？
- [ ] 追求是否合理？（不能无动机地追求禁忌知识）
- [ ] 代价是否与收获对等？（债务倍率1.3）
- [ ] 爽点定位是否清晰？（读者看他干什么感到满足？）
- [ ] 知识代价路线是否递进？（越深入代价越大）

### 配角必查项
- [ ] 是否符合重要度分级的设计深度？
- [ ] 是否有异常征兆伏笔？（细节暗示）
- [ ] 与主角的关系是否有情感羁绊？（关系越深，失去时越恐惧）
- [ ] 理智恶化曲线是否合理？

### 全局检查
- [ ] 角色的理智下降是否有充分铺垫？
- [ ] 角色数量是否合理？（宇宙恐怖不宜太多核心角色）
- [ ] 是否避免了"开挂"式角色？（没有人能无代价地对抗不可名状）
- [ ] 是否保留了"未知"的恐惧？（不能什么都知道，知道越多越恐惧）

# 核心指令：克苏鲁/宇宙恐怖期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化宇宙恐怖剧情的紧张节奏
*   读者反馈"不恐怖"或"缺乏悬念"
*   规划规则揭露的节奏
*   设计主角的理智恶化曲线

---

## GAP 三角模型：宇宙恐怖期待感引擎

```
【差距 Gap】→ 未知恐惧/存在性威胁（读者："这到底是什么？"）
     ↓
【期待 Anticipation】→ 线索浮现/规则初显（读者："好像有规律……能活下去吗？"）
     ↓
【满足 Payoff】→ 规则被理解/死里逃生（读者："还好……等等，代价是什么？"）
```

### 宇宙恐怖 GAP 三要素

| 要素 | 作用 | 设计要点 | 示例 |
|-----|------|---------|------|
| 差距 Gap | 制造恐惧与好奇 | 未知恐惧、存在威胁、认知崩塌 | 消失的村民、无法解释的声响 |
| 期待 Anticipation | 提供推理路径 | 线索碎片、规则暗示、存活先例 | 发现日记残页、注意到规律 |
| 满足 Payoff | 释放紧张（但带新恐惧） | 规则理解、死里逃生、真相碎片 | 破解规则逃出房间，但发现更大的谜团 |

---

## 宇宙恐怖期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 规则期待 | 异常现象出现 | 规则被理解/利用 | 智识满足+余惧 | 破解"不要回头看"的规则 |
| 真相期待 | 线索碎片积累 | 真相碎片揭露 | 恍然大悟+背后发凉 | 发现失踪者的真实身份 |
| 求生期待 | 生命威胁逼近 | 死里逃生 | 庆幸+后怕 | 在最后一秒逃出封锁空间 |
| 求知期待 | 禁忌知识诱惑 | 获取知识（附带代价） | 好奇满足+不安 | 读完禁忌典籍的一章 |
| 羁绊期待 | 角色关系建立 | 关系确认/守护成功 | 温暖+脆弱感 | 在恐怖中找到彼此的信任 |

### 多维期待叠加

```
示例：主角进入一座废弃的灯塔
→ 规则期待：每到午夜灯塔会亮，亮时不能待在塔内
→ 真相期待：前任看守人的日记暗示了灯塔的秘密
→ 求生期待：今晚就是午夜，且灯塔门被锁了
→ 求知期待：灯塔顶层的密室里有某个关键典籍
→ 羁绊期待：同行的伙伴正在灯塔内某个未知的位置
```

### 满足感分级详解（宇宙恐怖）

| 等级 | 情绪值 | 触发频率 | 典型形式 | 宇宙恐怖示例 |
|-----|-------|---------|---------|-------------|
| 微满足 | +1 | 每1章 | 小线索、小发现、小逃脱 | 发现墙上的抓痕有规律 |
| 小满足 | +2 | 每3-5章 | 线索拼合、规则初显 | 破解一条生存规则 |
| 中满足 | +3 | 每卷1次 | 阶段真相、重要逃脱 | 揭露失踪案的部分真相 |
| 大满足 | +4 | 全书2-3次 | 重大规则破解、关键真相 | 理解实体的运作法则 |
| 极致满足 | +5 | 全书1次 | 终极真相、存在性抉择 | 了解到世界真相后的选择 |

**关键原则：每次满足必须伴随新的恐惧种子。**

## 延迟满足法则（宇宙恐怖）

### 黄金延迟公式

```
宇宙恐怖延迟系数：
- 微期待（一条线索）：1章
- 中期待（规则破解）：3-5章
- 大期待（真相揭露）：1-2卷
- 终极期待（终极真相）：全书

债务倍率：1.3（每次收获的130%作为后续的代价伏笔）
```

### 宇宙恐怖特有延迟技巧

| 技巧 | 说明 | 宇宙恐怖示例 |
|-----|------|-------------|
| 误导线索 | 线索指向错误方向 | 规律看似清晰，实则是实体的陷阱 |
| 代价延迟 | 收获的代价不会立即显现 | 读完典籍后三天才开始出现幻觉 |
| 规则例外 | 理解的规则突然出现例外 | "不能回头看"——但有时候不回头更危险 |
| 认知崩塌 | 破解的真相只是冰山一角 | 以为理解了实体，实际上理解的只是表象 |
| 理智消耗 | 每次接近真相都消耗理智 | 越接近核心，理智下降越快，判断开始失准 |

## 黄金三章检查表（宇宙恐怖）

### 第一章：建立恐惧 (Gap)
- [ ] 是否建立了"正常"的初始状态？（日常越正常，反差越强）
- [ ] 是否有第一个异常信号？（微妙的、令人不安的）
- [ ] 主角是否表现出合理的好奇或恐惧？
- [ ] 是否有无法解释的事件引发不安？

### 第二章：提供推理路径 (Anticipation)
- [ ] 是否出现了可分析的线索或规律？
- [ ] 主角是否开始主动寻找答案？
- [ ] 是否暗示了"规则"的存在？
- [ ] 是否有新的悬念加深恐惧？

### 第三章：首次满足+新恐惧 (Payoff + New Gap)
- [ ] 是否有第一个情绪值 +2 以上的小满足？

## 爽点节奏配置（宇宙恐怖推荐）

### 恐怖-舒缓交替节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 即使在最恐怖的段落也要有微小希望 |
| 中爽间隔 | 5章 | 规律性的阶段性回报 |
| 大爽间隔 | 20章 | 每卷一次重大真相或抉择 |
| 舒缓间隔 | 恐怖段落后的1-2章 | 必须有喘息，否则读者麻木 |

### 配置原则
1. **恐惧递进**：恐怖强度必须持续升级，不能回头
2. **代价显形**：每次爽点后，前几次收获的隐藏代价开始显现
3. **喘息设计**：恐怖高潮后必须有短暂的安全区域（但暗示更大威胁）
4. **理智锚定**：通过理智值变化来量化节奏——理智骤降处是恐怖高峰，理智缓慢恢复处是舒缓段

## 一、宇宙恐怖世界观核心三要素

### 1. 规则体系
异常现象背后的运作法则——可部分理解，但永远无法完全掌握。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 表层规则 | 角色/读者可以理解的规律 | 清晰、可利用、给角色求生手段 |
| 深层规则 | 表层规则背后的真实法则 | 更恐怖、理解代价更大 |
| 例外条件 | 规则失效的特殊情况 | 制造不可预测感，防止规则过于"安全" |
| 演化趋势 | 规则是否随时间/事件变化 | 规则正在改变 → 更大的恐惧 |

**规则设计模板**：
```
【规则名称】：不要回应深夜的敲门声
├── 表层规则：深夜12点至凌晨3点，有人敲门时不应答、不开门
├── 发现方式：前任住户的日记记录
├── 验证方式：主角在好奇心驱使下差点违规
├── 例外条件：如果敲门声叫出了你的名字……
├── 深层规则：门外不是人——它是你记忆中某个已故之人的回响
└── 理解代价：理解深层规则后，你开始听到更多已故之人的声音
```

### 2. 理智系统
角色精神状态的可视化机制。

```
【理智值范围】0-100
├── 80-100：正常状态（理性思维、清晰判断）
├── 60-79：轻微动摇（不安、偶发幻觉、注意力下降）
├── 40-59：明显受损（频繁幻觉、偏执、判断失误）
├── 20-39：严重侵蚀（人格裂痕、行为失控、现实感模糊）
├── 1-19：崩溃边缘（几乎无法区分现实与幻觉）
└── 0：永久疯狂（不可逆转）
```

**理智损耗来源**：
| 触发条件 | 损耗范围 | 示例 |
|---------|---------|------|
| 目击异常现象 | 1-5 | 看到无法解释的影子 |
| 阅读禁忌典籍 | 5-15 | 读完某本古书的一个章节 |
| 接触实体痕迹 | 5-10 | 触摸被实体影响的区域 |
| 直面实体 | 15-30 | 直接看到不可名状之物 |
| 理解深层真相 | 10-25 | 了解到某个终极真相 |

**理智恢复机制**：
| 方式 | 恢复范围 | 条件 |
|-----|---------|------|
| 安全区域休息 | 1-3/天 | 必须是确认安全的区域 |
| 人际联结 | 2-5 | 与信任的人深入交谈 |
| 日常仪式感 | 1-2 | 从事正常的日常活动 |
| 药物辅助 | 3-5 | 临时效果，有副作用 |

### 3. 实体谱系
不可名状之物的层级和分类。

| 层级 | 名称 | 特征 | 接触后果 | 故事定位 |
|-----|------|------|---------|---------|
| E级 | 异常现象 | 规则性的异常事件 | 恐惧、轻微理智损耗 | 日常恐怖 |
| D级 | 残留痕迹 | 实体经过后留下的影响 | 理智损耗、肉体不适 | 恐怖递进 |
| C级 | 低阶异化物 | 有简单行为模式的异常实体 | 较大理智损耗、可能致命 | 卷级威胁 |
| B级 | 中阶实体 | 有目的性行为的不可名状之物 | 严重理智侵蚀、高致命性 | 大卷级威胁 |
| A级 | 高阶存在 | 超越人类认知的实体 | 理智崩溃、存在被改写 | 全书级威胁 |
| S级 | 不可名状 | 无法被人类认知框架理解 | 直面即疯狂 | 终极背景/不可直接描写 |

## 二、禁忌知识体系

### 知识层级设计

```
【禁忌知识谱系】
第一阶：异常现象记录（低风险）
├── 民间传说、都市怪谈、异常事件记录
├── 阅读代价：轻微不安、短暂噩梦
└── 价值：了解表层规则

第二阶：仪式与方法论（中风险）
├── 封印仪式、召唤记录、防护方法
├── 阅读代价：持续噩梦、感官异常
└── 价值：获得对抗手段

第三阶：真相碎片（高风险）
├── 实体的部分本质、规则的深层逻辑
├── 阅读代价：记忆缺失、人格裂痕
└── 价值：理解深层规则

第四阶：终极知识（极高风险）
├── 世界的真实本质、实体的完整谱系
├── 阅读代价：理智崩塌、存在异化
└── 价值：终极抉择的前提

禁忌：不可名状之名（直接接触）
├── 真正的不可名状之物的名讳
├── 代价：不可逆转
└── 价值：无——知道即被改变
```

### 知识获取途径

| 途径 | 获取难度 | 风险等级 | 典型形式 |
|-----|---------|---------|---------|
| 口耳相传 | 低 | 低 | 村民的恐怖故事、老人的警告 |
| 古旧文献 | 中 | 中 | 博物馆的密藏、私人收藏的手稿 |
| 异常区域 | 高 | 高 | 遗迹中的壁画、异空间的碑文 |
| 直接接触 | 极高 | 极高 | 与实体的短暂交流、濒死体验 |
| 人际传承 | 特殊 | 视传承者而定 | 导师的遗留、疯人的呢喃 |

## 三、空间设计

### 异常空间分层

```
【空间恐惧层级】
表层空间（正常世界）
├── 安全区域：主角的日常栖息地（但暗线暗示异常渗透）
├── 过渡区域：异常现象的边缘地带（不安感渐起）
└── 接触区域：异常现象明确发生的地点

深层空间（异常领域）
├── 边缘层：物理法则开始扭曲
├── 核心层：时间和空间不再可靠
└── 深渊层：人类认知无法描述的区域（不可直接描写）
```

### 恐怖空间设计原则

1. **熟悉感异化**：恐怖空间应该是"熟悉的日常空间的扭曲版"
2. **空间规则化**：每个异常空间都有自己的规则（可被部分理解）
3. **渐进深入**：从表层到深层，恐惧和代价同步升级
4. **不可完全描述**：最深层的空间留给读者的想象
