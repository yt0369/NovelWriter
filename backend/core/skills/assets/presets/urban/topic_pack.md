# 核心指令：都市重生角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 都市重生示例 |
|-----|------|-------------|
| 出身 | 家世、阶层、职业 | 贫民窟孤儿、破产商人、普通白领 |
| 前世经历 | 重生前的关键事件 | 被合伙人背叛、被妻子出轨、事业失败 |
| 重生优势 | 重生带来的核心优势 | 未来记忆、商业预判、人脉信息 |
| 创伤/执念 | 促成行为模式的心理锚点 | "绝不再被人踩在脚下"、"这辈子要守护家人" |

**底色公式**：
```
底色 = 前世失败经历 + 重生优势 → 形成的行事风格
```

**都市重生示例**：
- 主角底色：前世被合伙人背叛破产 + 重生回到十年前 → "精准布局、绝不信任外人"的行事风格
- 反派底色：富二代 + 前世靠家族碾压主角 → "目中无人、依赖家族"的人格

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 财富型 | 追求财富自由、商业帝国 | 赚钱、投资成功、商业碾压 |
| 复仇型 | 报复前世仇人 | 仇人破产、真相大白 |
| 弥补型 | 弥补前世遗憾 | 守护家人、挽回错过的人 |
| 权力型 | 追求社会地位、政商关系 | 地位提升、人脉扩张 |
| 守护型 | 保护家人、爱人 | 以弱胜强、守护成功 |

**追求的三个层次**：
```
表层追求：具体目标（赚到第一个亿、打败某个对手）
深层追求：身份认同（证明自己、被认可、超越前世）
终极追求：存在意义（守护家人、改变命运、活出不一样的人生）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 重生记忆 | 未来信息差 | 股市走向、房价趋势、行业风口 |
| 商业眼光 | 前世积累的商业经验 | 投资判断、谈判技巧、管理能力 |
| 人脉信息 | 知道谁会成功/失败 | 提前结交贵人、避开小人 |
| 技术预知 | 知道未来技术趋势 | 提前布局互联网、移动支付等 |
| 性格优势 | 前世磨练的内在优势 | 冷静、果断、隐忍、识人 |

**都市重生手段设计原则**：
1. **有限制**：重生记忆不是万能的（记忆模糊、蝴蝶效应）
2. **有成长**：手段要能升级（从小生意到商业帝国）
3. **有特色**：避免同质化（不是所有重生者都炒股买房）

## 二、都市重生爽点定位设计

> 都市重生角色的核心价值：**读者看他赚钱、打脸、逆袭爽不爽？**

### 爽点类型矩阵（都市重生特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **精准预判** | 利用未来信息精准投资 | 佩服、过瘾 | 商业天才、投资之神 |
| **打脸装逼** | 低调出手，震惊全场 | 痛快、解气 | 扮猪吃虎、隐形富豪 |
| **财富碾压** | 用钱解决问题 | 爽快、满足 | 从穷到富、财富自由 |
| **复仇打脸** | 让前世仇人付出代价 | 解气、痛快 | 重生复仇、以牙还牙 |
| **人脉布局** | 提前结交未来大佬 | 满足感 | 人脉经营、贵人相助 |
| **守护成功** | 保护前世失去的人 | 感动、温暖 | 守护家人、弥补遗憾 |

### 主角爽点定位模板（都市重生）

```
【爽点定位】
- 主爽点：精准预判 + 打脸装逼
- 辅爽点：财富碾压
- 爽点触发条件：被轻视/挑衅时，或商业决策时
- 爽点表现形式：平时低调，关键时刻展现惊人商业眼光

【重生优势】
- 核心信息差：未来10年的商业趋势
- 关键时间节点：股市暴涨、房价起飞、互联网风口
- 人脉记忆：知道谁会成为大佬

【财富路线】
- 短期目标：赚到第一桶金（100万）
- 中期目标：建立商业版图（1亿）
- 长期目标：成为行业巨头（100亿）
```

# 核心指令：都市重生期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化都市剧情的爽点节奏
*   读者反馈"不够爽"或"平淡"
*   规划商业布局前的铺垫
*   设计主角财富成长曲线

---

## GAP 三角模型：都市重生期待感引擎

```
【差距 Gap】→ 身份差距/财富差距（读者："这也太穷了"）
     ↓
【期待 Anticipation】→ 重生记忆/商业机会（读者："有戏！能翻身！"）
     ↓
【满足 Payoff】→ 赚到大钱/打脸仇人（读者："爽！终于翻身了..."）
```

### 都市重生 GAP 三要素

| 要素 | 作用 | 都市设计要点 | 示例 |
|-----|------|-------------|------|
| 差距 Gap | 制造不满足感 | 身份差距、财富差距、地位差距 | 穷小子 vs 富二代 |
| 期待 Anticipation | 提供解决路径 | 重生记忆、商业机会、贵人出现 | 知道某只股票要暴涨 |
| 满足 Payoff | 释放期待 | 赚到大钱、打脸仇人、地位反转 | 从穷小子变成亿万富翁 |

---

## 都市重生期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 逆袭期待 | 身份差距、被羞辱 | 财富碾压、地位反转 | 解气、痛快 | 穷小子变富豪 |
| 复仇期待 | 前世被背叛、被陷害 | 仇人破产、真相大白 | 快意、解恨 | 让前世仇人付出代价 |
| 财富期待 | 知道商业机会 | 投资成功、赚到大钱 | 兴奋、满足 | 精准投资暴涨股票 |
| 弥补期待 | 前世遗憾、失去的人 | 挽回遗憾、守护成功 | 感动、温暖 | 救回前世去世的亲人 |
| 人脉期待 | 知道谁会成功 | 结交大佬、获得支持 | 期待、满足 | 提前结交未来首富 |

### 多维期待叠加

```
示例：主角重生回到被公司开除的那天
→ 逆袭期待：从被开除到成为行业巨头
→ 复仇期待：让开除自己的老板破产
→ 财富期待：利用未来信息赚到第一桶金
→ 弥补期待：这次不再错过妻子
```

## 延迟满足法则（都市重生）

### 黄金延迟公式

```
都市延迟系数参考：
- 微期待（小谈判）：1章
- 中期待（投资回报）：3-5章

### 都市特有延迟技巧

| 技巧 | 说明 | 都市示例 |
|-----|------|---------|
| 资金不足 | 有信息但没钱 | 知道股票要涨但没有启动资金 |
| 蝴蝶效应 | 重生改变了部分历史 | 某个关键事件因为主角介入而改变 |
| 竞争对手 | 有人抢先一步 | 另一个重生者/商业天才出现 |
| 信任危机 | 没人相信主角的判断 | 投资建议被所有人否定 |

# 核心指令：都市重生爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 都市重生爽点三级分类

### 小爽点（+1~+2）
**定义**：小收获、小胜利、小打脸

**典型场景**：
- 谈判中占据上风
- 小额投资获利
- 打脸看不起自己的小人物
- 获得某个人的认可
- 发现一个商业机会
- 成功说服一个合作伙伴
- 前世仇人遇到小麻烦

**频率建议**：每1章一次

**作用**：维持快节奏爽感，让读者感受到持续进展

---

### 中爽点（+3~+4）
**定义**：阶段性商业成功、重要打脸

**典型场景**：
- 投资大获成功（翻倍以上）
- 商业布局阶段性完成
- 打脸富二代/商业对手
- 获得重要人脉支持
- 社会地位大幅提升
- 前世仇人遭受重大损失
- 成功挽回前世失去的人

**频率建议**：每5章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：重大商业胜利、终极打脸

**典型场景**：
- 成为行业领袖
- 前世仇人彻底破产
- 真实身份/实力全面揭晓
- 完成终极商业布局
- 成为首富/行业巨头
- 前世遗憾完全弥补

## 爽点节奏配置（都市重生推荐）

### 超高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 几乎每章都有进展 |
| 中爽间隔 | 5章 | 频繁打脸和商业成功 |
| 大爽间隔 | 20章 | 每卷一次大高潮 |

### 配置原则
1. **打脸不断**：最长不超过2章无打脸/赚钱
2. **财富可见**：让读者看到具体的财富数字增长
3. **复仇递进**：仇人的损失要逐步加大
4. **避免审美疲劳**：不能每章都是同样的打脸模式

# 核心指令：都市重生世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要构建都市重生的社会背景和商业格局
*   需要设计势力关系和人脉网络
*   需要规划主角的商业版图和经济路线
*   世界观设定出现矛盾或不符合现实逻辑

---

## 一、都市世界观核心三要素

### 1. 商业格局
故事发生的商业环境和经济脉络。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 时代背景 | 重生回到的具体年份 | 对应真实的经济发展节点（如2000年初互联网泡沫、2008金融危机） |
| 行业格局 | 核心产业和商业版图 | 房地产、互联网、金融、制造业等行业的分布 |
| 商业机会 | 重生者可利用的机会 | 股市走势、房价趋势、行业风口、企业兴衰 |
| 资本层级 | 从小资本到大资本的路径 | 第一桶金→千万→亿→百亿的合理路径 |

### 2. 社会阶层
都市社会的层级结构。

```
顶层：政商巨头、世家大族（主角的终极对手）
      ↓
上层：上市公司CEO、地方富豪（中期目标）
      ↓
中层：中小企业主、高级白领（起点或过渡）
      ↓
底层：打工人、小生意人（主角重生起点）
```

**阶层设计原则**：
1. **可跨越**：每个阶层之间都有跨越路径
2. **有代价**：跨越阶层需要付出努力和承担风险
3. **有信息差**：不同阶层掌握不同的信息（主角重生优势）
4. **蝴蝶效应**：主角的介入会改变部分人的命运

### 3. 势力网络
人与人之间的关系和权力结构。

| 关系类型 | 说明 | 剧情作用 |
|-----|------|---------|
| 血缘关系 | 家族、宗族 | 提供后盾或束缚 |
| 利益联盟 | 商业合作、互惠互利 | 可以结盟也可以翻脸 |
| 上下级 | 老板-员工、保护伞-被保护者 | 权力依附和利用 |
| 暗势力 | 黑白两道、灰色地带 | 增加都市世界的复杂度 |

---

## 二、都市时间线设计

### 经济发展节点（根据重生时间定制）

```
【时间线模板】
重生起点（如2005年）
├── 短期机会（1年内）：股市波动、小生意机会
├── 中期机会（3-5年）：房价起飞、互联网兴起
├── 长期机会（5-10年）：移动互联网、新兴行业
└── 关键事件：金融危机、政策变化、行业洗牌
```

### 蝴蝶效应设计

| 原史事件 | 主角介入后 | 影响范围 |
|---------|----------|---------|
| 某企业正常发展 | 主角提前投资 | 该企业加速发展 |
| 某人正常成功 | 主角提前结交 | 获得强力盟友 |
| 某人正常失败 | 主角出手相助 | 收服忠实追随者 |

**蝴蝶效应原则**：
- 主角只能影响与他有交集的人和事
- 改变越大，后续影响越不可控
- 不能改变宏观历史走向（如国家政策）

├── 科技园：互联网公司、创业孵化器

├── 新开发区：房地产、新兴产业

### 人脉网络模板

第三圈（资源）：政府关系、行业人脉
第四圈（对手）：商业对手、前世仇人
第五圈（暗线）：灰色人脉、地下势力
```

## 四、财富体系设计

### 资本阶段

| 阶段 | 资金量级 | 典型手段 | 社会地位 |
|-----|---------|---------|---------|
| 起步期 | 10万-100万 | 信息差、小生意 | 普通人 |
| 积累期 | 100万-1000万 | 投资、小公司 | 小老板 |
| 发展期 | 1000万-1亿 | 实业、金融 | 富人圈 |
| 扩张期 | 1亿-10亿 | 并购、上市 | 企业家 |
| 帝国期 | 10亿+ | 跨行业、资本运作 | 商业巨头 |

### 财富来源合理性检查

- [ ] 第一桶金的来源是否合理？（不能无中生有）
- [ ] 资金增长速度是否有逻辑支撑？
- [ ] 投资回报率是否符合当时的市场情况？
- [ ] 是否考虑了税收、法律等现实因素？

## 五、世界观设计检查清单

- [ ] 时代背景是否具体且真实？（具体到年份和事件）
- [ ] 商业格局是否合理？（行业分布、资本流向）
- [ ] 社会阶层是否清晰？（从底层到顶层的路径）
- [ ] 势力关系是否复杂但可理解？
- [ ] 蝴蝶效应是否有边界？（不能无限制改变历史）
- [ ] 财富路线是否合理？（不能一步登天）
- [ ] 都市细节是否到位？（地名、行业术语、时代特征）
- [ ] 世界观是否为剧情服务（而非为设定而设定）？
