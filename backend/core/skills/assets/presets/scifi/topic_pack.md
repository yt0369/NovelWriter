# 核心指令：科幻末世角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 科幻末世示例 |
|-----|------|-------------|
| 出身 | 末世前身份、末世后处境 | 前科学家、军人、普通幸存者、实验体 |
| 核心经历 | 末世中的关键事件 | 亲眼目睹城市毁灭、在废墟中独自求生、被组织抛弃 |
| 创伤/执念 | 促成行为模式的心理锚点 | "活下去"、"重建文明"、"找到家人" |

**底色公式**：
```
底色 = 末世前身份 + 末世经历 → 形成的生存哲学
```

**科幻末世示例**：
- 主角底色：前军事工程师 + 基地被丧尸攻破全灭 → "冷静理性、用科技武装自己"
- 反派底色：前政府高官 + 末世后建立独裁据点 → "为了生存不择手段、视人命为资源"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 生存型 | 活下去、保护同伴 | 绝境求生、化解危机 |
| 重建型 | 重建文明、建立基地 | 基地升级、科技突破 |
| 探索型 | 探索末世真相 | 发现新区域、揭开灾难真相 |
| 进化型 | 追求变异进化 | 能力觉醒、进化升级 |
| 复仇型 | 向造成末世的人复仇 | 真相大白、仇人伏诛 |

**追求的三个层次**：
```
表层追求：具体目标（找到安全的基地、获得食物和水）
深层追求：身份认同（成为幸存者的领袖、证明人类不会灭亡）
终极追求：存在意义（重建文明、拯救人类、揭开末世真相）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 变异能力 | 末世后觉醒的特殊能力 | 精神力、元素控制、身体强化 |
| 科技装备 | 末世前/后的科技产物 | 动力装甲、能量武器、纳米修复 |
| 生存技能 | 末世中磨练的技能 | 野外求生、战斗技巧、医疗急救 |
| 资源积累 | 掌握的物资和据点 | 食物储备、武器弹药、安全基地 |
| 领导力 | 组织和管理能力 | 团队建设、战术指挥、资源分配 |

**科幻末世手段设计原则**：
1. **有限制**：能力有副作用（变异不稳定、装备需要能源）
2. **有成长**：能力可以进化升级（变异等级、科技树）
3. **有代价**：使用能力要消耗资源（弹药有限、体力消耗）
4. **有环境**：能力受环境影响（辐射区、电磁干扰区）

---

## 二、科幻末世爽点定位设计

> 科幻末世角色的核心价值：**读者看他求生、建设、探索爽不爽？**

### 爽点类型矩阵（科幻末世特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **绝境求生** | 在极端环境中存活 | 紧张→释放 | 求生专家、末世老兵 |
| **科技碾压** | 用科技装备碾压敌人 | 痛快、过瘾 | 科技流、装备党 |
| **基地建设** | 从零建立安全基地 | 成就感 | 建设流、种田流 |
| **能力觉醒** | 变异能力觉醒/进化 | 兴奋、期待 | 进化流、异能者 |
| **探索发现** | 发现新区域/末世真相 | 好奇、震撼 | 探索流、真相追寻者 |
| **团队协作** | 带领团队完成任务 | 热血、感动 | 领袖型、团队核心 |

### 主角爽点定位模板（科幻末世）

```

- 主爽点：科技碾压 + 基地建设
- 辅爽点：能力觉醒
- 爽点触发条件：遭遇危机时、科技突破时
- 爽点表现形式：用科技装备碾压变异兽，基地不断升级壮大

【末世设定】
- 末世类型：[丧尸/变异兽/外星入侵/资源枯竭]
- 特殊能力：
- 科技水平：
- 当前据点：

【成长路线】
- 短期目标：建立安全据点、获得基础资源
- 中期目标：发展科技、扩大势力范围
- 长期目标：重建文明/揭开末世真相
```

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化末世剧情的爽点节奏
*   读者反馈"不够紧张"或"平淡"
*   规划科技突破前的铺垫
*   设计基地建设成长曲线

---

## GAP 三角模型：科幻末世期待感引擎

```
【差距 Gap】→ 生存危机/科技差距（读者："这也太危险了"）
     ↓
【期待 Anticipation】→ 发现资源/科技线索（读者："有戏！能突破！"）
     ↓
【满足 Payoff】→ 危机解除/科技突破（读者："爽！终于安全了..."）
```

### 科幻末世期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 生存期待 | 资源耗尽、敌人来袭 | 危机解除、获得资源 | 紧张→释放 | 丧尸围城后突围 |
| 科技期待 | 发现科技线索 | 科技突破、新装备 | 兴奋、期待 | 研发出能量护盾 |
| 建设期待 | 基地需要升级 | 基地扩建、防御升级 | 成就感 | 基地从帐篷变堡垒 |
| 探索期待 | 发现未知区域 | 揭开末世真相 | 好奇、震撼 | 发现灾难的真正原因 |
| 进化期待 | 变异能力不稳定 | 能力进化、稳定控制 | 兴奋、满足 | 变异能力升级 |

---

# 核心指令：科幻末世爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 科幻末世爽点三级分类

### 小爽点（+1~+2）
**定义**：小收获、小胜利

**典型场景**：
- 找到一批食物/水/弹药
- 击杀小型变异兽
- 成功修复一件装备
- 发现一个安全的临时据点
- 救下一个幸存者
- 变异能力小幅提升
- 成功制作一件简易武器

**频率建议**：每3章一次

**作用**：维持生存紧张感中的小满足

---

### 中爽点（+3~+4）
**定义**：重要突破、阶段胜利

**典型场景**：
- 科技突破（研发新武器/装备）
- 基地升级（新增防御设施/生产线）
- 击退一次中等规模进攻
- 发现重要的末世线索
- 变异能力大幅进化
- 成功探索新区域
- 获得大量稀缺资源

**频率建议**：每12章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：重大胜利、终极突破

**典型场景**：
- 击退大规模入侵
- 重大科技突破（改变战局的发明）
- 揭开末世真相的关键部分
- 基地从据点升级为城市
- 变异能力终极进化
- 消灭区域BOSS级威胁
- 与其他幸存者势力联盟

**频率建议**：每40章一次

**作用**：全书记忆点，读者口碑传播点

---

## 爽点节奏配置（科幻末世推荐）

### 中频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 3章 | 持续的小收获 |
| 中爽间隔 | 12章 | 科技突破/基地升级 |
| 大爽间隔 | 40章 | 重大胜利 |

### 配置原则
1. **危机不断**：安全期不超过5章就要有新威胁
2. **资源紧张**：始终保持资源稀缺感
3. **科技递进**：科技突破要有明确的等级感
4. **探索驱动**：用未知区域驱动剧情前进

## 一、末世核心三要素

### 1. 末世成因
灾难的起源决定了世界的基本规则。

| 类型 | 说明 | 世界特征 |
|-----|------|---------|
| 病毒末世 | 病毒导致丧尸/变异 | 感染区、安全区、免疫者 |
| 外星入侵 | 外星生物入侵地球 | 占领区、抵抗区、科技差距 |
| 资源枯竭 | 能源/食物耗尽 | 资源争夺、社会崩溃 |
| 自然灾变 | 气候剧变/地质灾害 | 极端环境、生存挑战 |
| 科技失控 | AI叛变/实验失败 | 科技禁区、机械敌人 |

### 2. 科技体系
末世中的科技水平和发展方向。

**科技等级**：
```
Lv.1 原始科技：冷兵器、简易工具
Lv.2 基础科技：火器、基础电力
Lv.3 中级科技：自动武器、太阳能
Lv.4 高级科技：能量武器、动力装甲
Lv.5 尖端科技：纳米技术、基因改造
Lv.6 超级科技：反重力、空间折叠
```

### 3. 变异体系
末世中的变异生物和能力者。

**变异等级**：
| 等级 | 威胁度 | 特征 | 对应科技 |
|-----|-------|------|---------|

## 二、资源与势力体系

### 核心资源

| 资源 | 稀缺度 | 获取方式 | 冲突点 |
|-----|-------|---------|-------|
| 食物/水 | 高 | 种植/净化/搜刮 | 基础生存争夺 |
| 弹药/武器 | 中高 | 制造/搜刮 | 军事实力 |
| 能源 | 高 | 太阳能/核能/晶石 | 科技发展瓶颈 |
| 医疗用品 | 极高 | 制造/搜刮 | 生死攸关 |
| 变异材料 | 中 | 击杀变异兽 | 科技升级材料 |

### 势力格局

```
军事据点（武力强、资源多、纪律严）
  ↕ 贸易/冲突
幸存者聚落（人数多、资源少、组织松散）
  ↕ 依附/对抗
科研基地（科技强、人数少、需要保护）
  ↕ 合作/利用
流浪团队（机动性强、资源不稳定）
```
