# 核心指令：悬疑推理角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 悬疑推理示例 |
|-----|------|-------------|
| 出身 | 职业、背景、专业 | 退休刑警、法医、心理学教授、记者 |
| 核心经历 | 塑造性格的关键事件 | 亲人被害悬案未破、目睹冤案、自己曾被冤枉 |
| 创伤/执念 | 促成行为模式的心理锚点 | "真相必须大白"、"不能让无辜者蒙冤" |

**底色公式**：
```
底色 = 专业背景 + 核心经历 → 形成的执念/行事风格
```

**悬疑推理示例**：
- 侦探底色：退休刑警 + 女儿被害悬案未破 → "偏执追求真相、不放过任何细节"
- 反派底色：高智商罪犯 + 童年被虐待 → "精密犯罪、享受与侦探的智力博弈"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 真相型 | 追求案件真相 | 真相揭晓、逻辑闭环 |
| 正义型 | 追求公平正义 | 罪犯伏法、冤案昭雪 |
| 复仇型 | 为亲人追查真凶 | 真凶落网、真相大白 |
| 挑战型 | 享受推理的过程 | 破解谜题、智力碾压 |
| 救赎型 | 弥补过去的错误 | 阻止悲剧、拯救无辜 |

**追求的三个层次**：
```
表层追求：具体目标（破解这个案件、找到凶手）
深层追求：身份认同（证明自己的推理能力、不再让悲剧重演）
终极追求：存在意义（追求绝对的真相、守护正义）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 推理能力 | 逻辑推理和分析 | 演绎法、归纳法、排除法 |
| 观察力 | 细节捕捉能力 | 微表情识别、现场痕迹分析 |
| 专业知识 | 特定领域的专业能力 | 法医学、心理学、犯罪学、化学 |
| 人脉资源 | 可调用的关系网 | 警方关系、线人、专家顾问 |
| 心理素质 | 面对危险的冷静 | 临危不乱、高压下思考 |

**悬疑推理手段设计原则**：
1. **有盲点**：侦探不是全知全能的（有知识盲区、有情感干扰）
2. **有对手**：罪犯也要有智慧（不能太容易被抓）
3. **有代价**：追查真相要付出代价（危险、失去信任、心理创伤）
4. **有逻辑**：推理过程必须逻辑自洽

## 二、悬疑推理爽点定位设计

> 悬疑推理角色的核心价值：**读者看他推理、发现线索、揭露真相爽不爽？**

### 爽点类型矩阵（悬疑推理特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **逻辑碾压** | 用严密逻辑推翻嫌疑人的谎言 | 佩服、过瘾 | 高智商侦探 |
| **线索串联** | 将零散线索串联成完整链条 | 恍然大悟 | 细节控、拼图大师 |
| **真相反转** | 揭露出人意料的真相 | 震撼、惊叹 | 真相追寻者 |
| **心理博弈** | 与罪犯的心理攻防 | 紧张、刺激 | 心理分析师 |
| **绝境逃生** | 在危险中脱身 | 紧张→释放 | 冒险型侦探 |
| **冤案昭雪** | 为无辜者洗清冤屈 | 感动、解气 | 正义守护者 |

## 三、悬疑推理特有角色类型

### 嫌疑人设计要点

每个嫌疑人都需要：
1. **作案动机**：为什么要杀人/犯罪
2. **作案能力**：有没有能力实施犯罪
3. **不在场证明**：是否有alibi
4. **隐藏秘密**：即使不是凶手也有秘密

**嫌疑人设计模板**：
```
- 姓名：
- 与受害者关系：
- 作案动机（强度：强/中/弱）：
- 作案能力（有/无）：
- 不在场证明（有/无/可疑）：
- 隐藏秘密：
- 可疑行为：
- 是否为真凶：
- 红鲱鱼设计（如果不是真凶）：
```

### 红鲱鱼（误导线索）设计

| 类型 | 说明 | 示例 |
|-----|------|------|
| 假嫌疑人 | 看起来最可疑但不是凶手 | 有动机有能力但有完美alibi |
| 假线索 | 指向错误方向的线索 | 凶器上的指纹是被栽赃的 |
| 假动机 | 看似合理但不是真正动机 | 以为是情杀其实是灭口 |
| 假时间线 | 误导时间判断 | 尸体被移动导致死亡时间判断错误 |

# 核心指令：悬疑推理期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化悬疑剧情的爽点节奏
*   读者反馈"不够紧张"或"猜到了"
*   规划真相揭晓前的铺垫
*   设计线索释放节奏

---

## GAP 三角模型：悬疑推理期待感引擎

```
【差距 Gap】→ 信息差距/真相未知（读者："到底是谁干的？"）
     ↓
【期待 Anticipation】→ 线索出现/推理推进（读者："快了！要破案了！"）
     ↓
【满足 Payoff】→ 真相揭晓/逻辑闭环（读者："原来如此！太精彩了..."）
```

### 悬疑推理 GAP 三要素

| 要素 | 作用 | 悬疑设计要点 | 示例 |
|-----|------|-------------|------|
| 差距 Gap | 制造悬念 | 谜团、疑点、矛盾 | 密室杀人、不可能犯罪 |
| 期待 Anticipation | 推进推理 | 线索发现、嫌疑人排除 | 发现关键物证 |
| 满足 Payoff | 揭晓真相 | 逻辑闭环、真相震撼 | 凶手身份揭晓 |

---

## 悬疑推理期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 悬念期待 | 谜团、疑点 | 真相揭晓 | 好奇→震撼 | 密室之谜的解答 |
| 推理期待 | 线索积累 | 逻辑闭环 | 期待→恍然大悟 | 所有线索串联成链 |
| 危险期待 | 侦探陷入危险 | 脱险成功 | 紧张→释放 | 被凶手盯上后脱身 |
| 正义期待 | 无辜者被冤枉 | 冤案昭雪 | 愤怒→解气 | 真凶落网、冤者清白 |
| 博弈期待 | 侦探vs罪犯 | 智力碾压 | 紧张→佩服 | 审讯中攻破心理防线 |

### 多维期待叠加

```
示例：发现第二具尸体，凶手仍在逍遥法外
→ 悬念期待：凶手到底是谁？
→ 危险期待：侦探会不会成为下一个目标？
→ 推理期待：两个案件之间有什么联系？
→ 正义期待：必须在凶手再次作案前抓住他
```

## 线索释放节奏

### 线索分级

| 级别 | 作用 | 释放时机 | 示例 |
|-----|------|---------|------|
| A级线索 | 直接指向真相 | 全书后1/3 | 凶手的DNA |
| B级线索 | 缩小范围 | 全书中段 | 排除某个嫌疑人 |
| C级线索 | 提供方向 | 全书前半 | 作案动机的暗示 |
| D级线索 | 制造氛围 | 随时 | 不安的细节 |
| 红鲱鱼 | 误导方向 | 前2/3 | 假线索、假嫌疑人 |

### 线索释放原则
1. **公平原则**：所有关键线索必须在真相揭晓前呈现给读者
2. **递进原则**：线索从模糊到清晰，从外围到核心
3. **误导原则**：红鲱鱼要在真相揭晓前被排除
4. **回收原则**：所有线索最终都要有解释

# 核心指令：悬疑推理爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 悬疑推理爽点三级分类

### 小爽点（+1~+2）
**定义**：发现线索、推翻谎言

**典型场景**：
- 发现一个矛盾的证词
- 在现场找到被忽略的痕迹
- 推翻嫌疑人的一个谎言
- 发现两个事件之间的联系
- 侦探的一个精彩推理
- 排除一个不可能的假设
- 发现一个新的疑点

**频率建议**：每3章一次

**作用**：维持悬念张力，让读者感受到推理在推进

---

### 中爽点（+3~+4）
**定义**：重大线索突破、嫌疑人排除

**典型场景**：
- 发现关键物证
- 排除一个主要嫌疑人
- 推翻一个看似完美的不在场证明
- 发现案件之间的关联
- 侦探在审讯中攻破心理防线
- 发现凶手的作案手法
- 一个重大的推理突破

**频率建议**：每10章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：真相揭晓、逻辑闭环

**典型场景**：
- 凶手身份揭晓
- 所有线索串联成完整链条
- 不可能犯罪的手法揭秘
- 隐藏的真正动机曝光
- 冤案昭雪
- 终极真相震撼揭晓

**频率建议**：全书1次（终极真相）

**作用**：全书最高潮，读者口碑传播点

---

## 爽点节奏配置（悬疑推理推荐）

### 长线铺垫节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 3章 | 持续发现线索 |
| 中爽间隔 | 10章 | 阶段性推理突破 |
| 大爽间隔 | 全书1次 | 终极真相揭晓 |

### 配置原则
1. **悬念不断**：最长不超过5章无新线索/疑点
2. **递进推理**：推理从外围到核心，逐步深入
3. **公平呈现**：关键线索必须在真相前呈现
4. **真相震撼**：终极真相要出人意料又合情合理

# 核心指令：悬疑推理世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计核心谜题和线索网络
*   需要规划真相层级和揭晓节奏
*   需要构建角色的秘密和隐藏关系
*   谜团逻辑出现漏洞或不合理

---

## 一、悬疑世界观核心三要素

### 1. 谜团体系
故事的核心谜题和结构。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 核心谜题 | 故事最大的悬念 | 必须足够吸引人，全书贯穿 |
| 子谜题 | 每卷/每案的小谜团 | 逐步推进，指向核心谜题 |
| 红鲱鱼 | 误导读者的假线索 | 至少2-3个，但不能太假 |
| 真相层级 | 真相的分层次揭晓 | 表面真相→深层真相→终极真相 |

**谜团层级设计**：
```
第一层（表面）：案件本身的表象（谁、什么、何时、何地）
    ↓
第二层（内层）：案件背后的动机和关联（为什么、怎么做到的）
    ↓
第三层（深层）：更大的阴谋或组织（谁在幕后操控）
    ↓
第四层（终极）：核心谜题的终极真相（颠覆认知的反转）
```

### 2. 线索网络
支撑谜团的线索系统。

| 线索类型 | 说明 | 放置方式 |
|---------|------|---------|
| 物证 | 实物证据 | 犯罪现场、搜查发现 |
| 人证 | 人物证言 | 对话中发现、审讯获得 |
| 行为线索 | 角色的异常行为 | 心理分析、行为模式 |
| 环境线索 | 场景中的细节 | 天气、时间、位置矛盾 |
| 文件线索 | 日记、信件、记录 | 遗物中发现、秘密文件 |

**线索布局原则**：
1. **公平性**：所有关键线索必须在揭晓前出现
2. **隐蔽性**：线索混在大量普通信息中
3. **关联性**：线索之间有逻辑联系
4. **误导性**：部分线索指向错误方向（红鲱鱼）

### 3. 角色秘密系统

| 秘密类型 | 说明 | 揭晓时机 |
|---------|------|---------|
| 身份秘密 | 真实身份与表面身份不符 | 中后期揭晓 |
| 过去秘密 | 角色隐藏的过去 | 逐步揭示 |
| 动机秘密 | 真实的动机被隐藏 | 关键时刻揭晓 |
| 关系秘密 | 隐藏的人际关系 | 制造反转 |
| 能力秘密 | 隐藏的特殊能力 | 危机时刻展现 |

## 二、悬疑世界观结构

### 舞台设定

```
【场景模板】
核心场景（主要事件发生地）
├── 主场景：犯罪/事件现场
├── 调查场景：可搜集线索的区域
├── 秘密场景：隐藏真相的地方
└── 对峙场景：最终揭晓的舞台

辅助场景
├── 安全区域：主角思考和分析的地方
├── 社交场所：获取情报的渠道
└── 危险区域：主角可能遇险的地方
```

### 组织与势力

| 类型 | 说明 | 剧情作用 |
|-----|------|---------|
| 调查组织 | 警方/侦探社/调查局 | 提供调查框架 |
| 神秘组织 | 幕后黑手的组织 | 制造更大的阴谋 |
| 利益集团 | 涉案的各方势力 | 制造复杂的动机 |
| 帮助者网络 | 主角的协助者 | 提供资源和信息 |

## 三、悬疑节奏设计

### 真相揭晓节奏

| 阶段 | 揭晓内容 | 读者情绪 | 典型位置 |
|-----|---------|---------|---------|
| 第一阶段 | 部分线索拼合 | 好奇、紧张 | 每案结束时 |
| 第二阶段 | 子谜题真相揭晓 | 满足、更多疑问 | 每卷结束时 |
| 第三阶段 | 大阴谋初步浮现 | 震惊、期待 | 全书50%处 |
| 第四阶段 | 终极真相揭晓 | 震撼、恍然大悟 | 全书高潮 |

### 悬疑世界特有节奏

```
线索发现 → 分析推理 → 遇到障碍 → 新线索 → 假说修正
→ 接近真相 → 反转（假真相）→ 真相大白
```
