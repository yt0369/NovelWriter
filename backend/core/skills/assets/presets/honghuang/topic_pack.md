# 核心指令：洪荒封神角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 洪荒封神示例 |
|-----|------|-------------|
| 出身 | 背景、起点 | 先天生灵、后天修士、转世重生、截教弟子、阐教弟子 |
| 核心经历 | 塑造性格的关键事件 | 师门被灭、量劫劫难、法宝被夺、封神榜上有名 |
| 创伤/执念 | 促成行为模式的心理锚点 | "证道成圣"、"渡过量劫"、"复兴师门" |

**底色公式**：
```
底色 = 洪荒起点 + 核心经历 → 形成的修道态度
```

**洪荒封神示例**：
- 主角底色：转世重生 + 前世被封神榜算计 → "谨慎低调、避开量劫"
- 对手底色：阐教嫡传 + 从小受圣人教导 → "骄傲自负、看不起截教"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 成圣型 | 证道成圣、超脱天道 | 成圣、超脱 |
| 量劫型 | 渡过量劫、避开封神 | 渡劫成功、逃脱封神 |
| 因果型 | 了结因果、偿还业力 | 因果了结、功德圆满 |
| 逍遥型 | 逍遥自在、不问世事 | 隐居修炼、自在逍遥 |
| 复仇型 | 复仇雪恨、重振师门 | 复仇成功、师门复兴 |

**追求的三个层次**：
```
表层追求：具体目标（突破境界、获得法宝、渡过某个劫难）
深层追求：身份认同（证明自己的道、被认可为大能）
终极追求：存在意义（证道成圣、改变洪荒格局、超脱天道）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 法力修为 | 境界、法力 | 金仙、太乙金仙、大罗金仙、准圣、圣人 |
| 法宝收集 | 先天灵宝、后天法宝 | 太极图、盘古幡、诛仙剑阵、混沌钟 |
| 因果业力 | 功德、业力 | 功德金身、业力缠身、因果报应 |
| 天道感悟 | 天道规则、大道法则 | 三千大道、天道规则、圣人之道 |
| 师门背景 | 师承、靠山 | 三清弟子、西方教徒、散修 |

**洪荒封神手段设计原则**：
1. **有短板**：不能所有方面都最强（法力强但法宝少）
2. **有成长**：修为要能提升（从金仙到圣人的成长线）
3. **有特色**：要有标志性的法宝/神通（独特的大道）
4. **有对手**：对手也要有特色和实力

## GAP 三角模型：洪荒封神期待感引擎

```
【差距 Gap】→ 境界差距/劫难难度（读者："能渡劫吗？"）
     ↓
【期待 Anticipation】→ 修炼提升/法宝准备（读者："有戏！准备好了！"）
     ↓
【满足 Payoff】→ 境界突破/劫难渡过（读者："爽！突破了！渡劫成功！"）
```

## 洪荒封神爽点三级分类

### 小爽点（+1~+2）
**定义**：小境界突破、法宝获得

**典型场景**：
- 获得法宝（后天法宝）
- 小境界突破（金仙初期→中期）
- 击败敌人
- 获得机缘
- 炼制丹药/法宝
- 收徒/收小弟
- 打脸看不起自己的人
- 发现洞天福地

**频率建议**：每1章一次

**作用**：维持修炼感，让读者感受到持续成长

---

### 中爽点（+3~+4）
**定义**：大境界突破、先天灵宝获得

**典型场景**：
- 大境界突破（金仙→太乙金仙）
- 获得先天灵宝
- 击败强敌
- 量劫小胜
- 功德降临
- 收服强力坐骑
- 获得大神通
- 因果了结

**频率建议**：每5章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：证道成圣、量劫关键胜利

**典型场景**：
- 证道成圣
- 量劫关键胜利
- 改变洪荒格局
- 获得至宝（盘古幡、太极图）
- 证道大罗金仙
- 超脱天道
- 击败圣人/准圣

**频率建议**：每20章一次

**作用**：全书记忆点，读者口碑传播点

## 爽点节奏配置（洪荒封神推荐）

### 高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 每章都有修炼进展/法宝获得 |
| 中爽间隔 | 5章 | 频繁的境界突破/法宝获得 |
| 大爽间隔 | 20章 | 重大突破/量劫胜利 |

### 配置原则
1. **修炼不断**：最长不超过2章无修炼进展
2. **境界递进**：境界要逐步提升
3. **法宝成长**：法宝要有可见的提升
4. **量劫推进**：量劫要逐步推进

# 核心指令：洪荒封神世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要构建洪荒世界的基础设定
*   需要设计境界体系或法宝体系
*   需要规划三教格局或封神榜
*   世界观设定出现矛盾或漏洞

## 一、洪荒世界核心三要素

### 1. 洪荒时期

| 时期 | 说明 | 主要事件 |
|-----|------|---------|
| 混沌初开 | 盘古开天 | 盘古开天辟地、三千魔神陨落 |
| 龙汉量劫 | 三族争霸 | 龙凤麒麟三族争霸、道魔之战 |
| 巫妖量劫 | 巫妖大战 | 十二祖巫 vs 妖族天庭 |
| 封神量劫 | 封神之战 | 阐截两教大战、封神榜 |
| 西游量劫 | 西天取经 | 唐僧取经、佛教东传 |

### 2. 境界体系

**标准境界体系**：
```
凡人 → 地仙 → 天仙 → 真仙 → 玄仙 → 金仙 → 太乙金仙 → 大罗金仙 → 准圣 → 圣人
```

**境界设计原则**：
1. **层次清晰**：每个境界的实力差距要明确
2. **突破条件**：每个境界的突破条件不同
3. **寿命对应**：境界越高寿命越长
4. **战力倍率**：高一个大境界约10倍战力差

**境界设计模板**：
```
【境界名称】
- 层次划分：初期/中期/后期/巅峰
- 标志特征：（如金仙标志是不死不灭）
- 寿命上限：
- 战力范围：
- 突破条件：
- 突破难度：
- 该境界的社会地位：
```

### 3. 天道规则

| 规则 | 说明 | 剧情作用 |

## 二、三教体系

### 教派划分

| 教派 | 教主 | 教义 | 弟子特点 |
|-----|------|------|---------|
| 人教 | 太上老君 | 无为而治 | 少而精 |
| 阐教 | 元始天尊 | 顺天应人 | 精英教育 |
| 截教 | 通天教主 | 有教无类 | 数量众多 |
| 西方教 | 接引/准提 | 慈悲度化 | 虚伪贪婪 |

### 教派内部结构

**阐教结构**：
```
元始天尊
  ├─ 十二金仙（核心弟子）
  ├─ 三代弟子（金仙弟子的徒弟）
  └─ 四代弟子（三代弟子的徒弟）
```

**截教结构**：
```
通天教主
  ├─ 四大亲传（多宝、金灵、龟灵、无当）
  ├─ 外门弟子（万仙来朝）
  └─ 妖族弟子（各种妖怪）
```

### 教派关系设计

**核心矛盾来源**：
1. **教义冲突**：阐教精英 vs 截教平等
2. **资源争夺**：法宝、灵脉、弟子
3. **量劫算计**：封神榜上谁上榜
4. **圣人博弈**：圣人之间的算计

## 四、封神榜体系

### 封神榜规则

**封神榜作用**：
- 封神：将死去的修士封为神灵
- 限制：上榜者受天庭管辖
- 不死：上榜者不死不灭但失去自由

**上榜条件**：
1. 量劫中死亡
2. 有仙缘但未成正果
3. 被圣人算计

### 神位分配

| 神位 | 数量 | 地位 | 代表 |
|-----|------|------|------|
| 三清 | 3 | 最高 | 太上老君、元始天尊、通天教主 |
| 四御 | 4 | 极高 | 玉皇大帝、紫微大帝 |
| 五方五老 | 5 | 高 | 东方青帝、南方赤帝 |
| 六司 | 6 | 中 | 雷部、火部、瘟部 |
| 365正神 | 365 | 低 | 各路神仙 |

## 五、量劫体系

### 量劫类型

| 量劫 | 时期 | 主要矛盾 | 结果 |
|-----|------|---------|------|
| 龙汉量劫 | 上古 | 三族争霸 | 三族退出舞台 |
| 巫妖量劫 | 远古 | 巫妖大战 | 两败俱伤 |
| 封神量劫 | 上古 | 阐截大战 | 截教覆灭 |
| 西游量劫 | 近古 | 佛教东传 | 佛教兴盛 |

### 量劫规则

**量劫触发**：
- 天道定期清洗
- 因果业力积累
- 圣人算计

**量劫影响**：
- 大量修士陨落
- 格局重新洗牌
- 天道规则调整
