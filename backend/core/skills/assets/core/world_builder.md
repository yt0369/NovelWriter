# 核心指令：世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建或扩展世界观设定
*   剧情涉及新势力/新规则，但设定缺失
*   力量体系/社会结构需要细化
*   需要设计等级体系或晋升机制

---

## 一、核心原则：设定为剧情服务

> ⚠️ **拒绝维基百科式设定**：每一个设定都必须服务于冲突和爽点。

### 设定的三个层次

| 层次 | 内容 | 作用 |
|-----|------|------|
| **功能性设定** | 力量体系、等级划分 | 提供成长期待、爽点来源 |
| **氛围性设定** | 地理风貌、社会风貌 | 增强沉浸感、世界观质感 |
| **剧情性设定** | 核心矛盾、资源稀缺 | 驱动冲突、制造困境 |

---

## 二、力量体系设计

> 力量体系是男频网文的核心，直接关系到读者的期待感和爽点体验。

### 力量等级设计原则

| 原则 | 说明 | 重要性 |
|-----|------|--------|
| **等级清晰** | 每个等级有明确的能力差距 | ⭐⭐⭐⭐⭐ |
| **晋升可见** | 读者能看到主角的进步 | ⭐⭐⭐⭐⭐ |
| **越级可能** | 主角能越级挑战（爽点来源） | ⭐⭐⭐⭐ |
| **代价存在** | 获得力量需要付出代价 | ⭐⭐⭐ |
| **瓶颈设计** | 突破瓶颈是关键剧情点 | ⭐⭐⭐⭐ |

### 等级划分模板

```
【力量等级体系】

### 基础等级（常见套路）
| 等级 | 名称 | 特征 | 社会地位 |
|-----|------|------|---------|
| 1级 | [等级名] | [能力特征] | [底层/平民] |
| 2级 | [等级名] | [能力特征] | [中层/精英] |
| ... | ... | ... | ... |
| N级 | [等级名] | [能力特征] | [顶层/传说] |

### 突破机制
- 突破条件：
- 突破瓶颈：
- 突破失败后果：

### 越级挑战规则
- 正常情况下：X级打不过Y级
- 特殊情况下（金手指/秘法）：可越Z级挑战
```

### 常见力量体系类型

| 类型 | 典型题材 | 等级特征 | 爽点来源 |
|-----|---------|---------|---------|
| 修炼流 | 玄幻、仙侠 | 境界突破、寿元延长 | 境界碾压、逆天突破 |
| 系统流 | 都市、游戏 | 等级/技能点/任务 | 系统奖励、数值碾压 |
| 血脉流 | 异能、玄幻 | 血脉觉醒、形态变化 | 血脉爆发、隐藏血脉 |
| 科技流 | 科幻、末世 | 装备等级、义体强化 | 装备升级、科技碾压 |
| 职业流 | 西幻、游戏 | 职业等级、技能解锁 | 职业进阶、隐藏职业 |

## 三、核心矛盾驱动 (Conflict-Driven Setting)

### 世界级矛盾设计

| 矛盾类型 | 说明 | 剧情作用 |
|---------|------|---------|
| **资源稀缺** | 这个世界什么最珍贵？ | 制造争夺、驱动冲突 |
| **阶级固化** | 力量体系如何导致阶级差异？ | 建立差距(Gap)、提供逆袭空间 |
| **对立阵营** | 基于核心理念的冲突 | 提供外部冲突来源 |
| **生存威胁** | 外部威胁（魔族、天灾等） | 推动主角成长、团结力量 |

### 资源稀缺性设计

```
【核心资源设定】
- 资源名称：
- 稀缺程度：[极度稀缺/稀缺/一般]
- 获取方式：
- 主要用途：
- 争夺现状：
```

**示例**：
- 修仙世界：灵气/灵石（决定修炼速度）
- 末世世界：安全区/食物（决定生存）
- 都市世界：权力/金钱（决定地位）

## 四、势力格局设计

### 势力分层

| 层级 | 势力类型 | 特征 | 与主角关系 |
|-----|---------|------|-----------|
| **顶级势力** | 统治级 | 控制核心资源 | 远期目标/最终BOSS |
| **一流势力** | 区域霸主 | 拥有重要资源 | 中期对手/合作对象 |
| **二流势力** | 地方豪强 | 有一定资源 | 早期对手/踏脚石 |
| **三流势力** | 小组织 | 资源有限 | 早期剧情/背景板 |

### 势力关系矩阵

```
【势力关系图】

势力A ───[敌对]───> 势力B
  │                    │
  │[合作]              │[从属]
  ↓                    ↓
势力C <───[世仇]──── 势力D

关键：
- 敌对：核心利益冲突
- 合作：共同利益
- 从属：实力差距悬殊
- 世仇：历史恩怨
```

### 势力设计模板

```
【势力档案】

## 基础信息
- 势力名称：
- 势力类型：[宗门/家族/组织/国家]
- 实力等级：[顶级/一流/二流/三流]
- 地盘范围：

## 内部结构
- 领导者：
- 核心成员：
- 组织架构：

## 核心资源
- 拥有什么资源：
- 资源获取方式：

## 对外关系
- 敌对势力：
- 盟友势力：
- 从属势力：

## 与主角关系
- 初始关系：
- 关系演变：
- 关键事件：
```

## 五、细节颗粒度 (Granularity)

### 宏观设定
- 历史大事件（影响当前格局的历史）
- 版图结构（地理/政治分区）
- 时间线（重要事件的时间轴）

### 微观设定
- 衣食住行（日常生活细节）
- 货币购买力（价值体系）
- 俚语脏话（语言特色）
- 底层人民的生存状态（社会质感）

### 微观设定示例

```
【生活细节设定】

### 货币体系
- 基础货币：1银币 = 100铜币 = 一个普通人一天的生活费
- 高级货币：1金币 = 100银币
- 购买力参考：
  - 一顿普通饭菜：10铜币
  - 一把普通武器：5银币
  - 一本入门功法：10金币

### 日常用语
- 见面问候：[设定]
- 常见诅咒：[设定]
- 职业术语：[设定]
```

## 六、世界观的收敛性原则

> ⚠️ **核心原则**：能用已有设定解释的，不要新增设定。

### 设定优先级

```
处理矛盾时的优先级：
删除 > 合并 > 增加解释（尽量避免）
```

### 设定检查

在添加新设定前，先问自己：
1. 这个设定是否必要？（删掉会怎样？）
2. 能否用已有设定解释？
3. 这个设定会不会和已有设定冲突？
4. 这个设定是否为剧情服务？

---

## 检查清单

在完成世界观设定后，必须检查：

### 力量体系
- [ ] 等级是否清晰可理解？
- [ ] 突破机制是否明确？
- [ ] 越级挑战是否合理？
- [ ] 等级与爽点是否挂钩？

### 核心矛盾
- [ ] 资源稀缺性是否明确？
- [ ] 阶级差异是否清晰？
- [ ] 是否有足够的外部冲突来源？

### 势力格局
- [ ] 势力层级是否清晰？
- [ ] 势力关系是否合理？
- [ ] 是否为剧情留有空间？

### 收敛性
- [ ] 是否有冗余设定？
- [ ] 设定之间是否矛盾？
- [ ] 新设定是否可以用旧设定解释？
