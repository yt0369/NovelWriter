# 核心指令：玄幻修仙角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

底色决定角色的**行事风格**和**价值观**，是角色一切行为的底层逻辑。

| 维度 | 说明 | 玄幻修仙示例 |
|-----|------|-------------|
| 出身 | 家世、宗门、灵根 | 散修出身、世家嫡子、隐世宗门弟子 |
| 核心经历 | 塑造性格的关键事件 | 宗门被灭、被夺舍重生、意外获得传承 |
| 创伤/执念 | 促成行为模式的心理锚点 | "绝不再让宗门覆灭"、"必证长生大道" |

**底色公式**：
```
底色 = 出身 + 核心经历 → 形成的执念/行事风格
```

**玄幻修仙示例**：
- 主角底色：散修出身 + 被正道宗门欺压 → "隐忍发育、一击必杀"的行事风格
- 反派底色：魔道世家 + 曾被正道围剿 → "极端仇视正道、不择手段"的人格

---

### 2. 追求（Drive）── 他要去哪里？

追求是角色的**行动引擎**，玄幻修仙角色的追求通常围绕修炼、复仇、长生。

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 长生型 | 追求长生不老、证道成仙 | 境界突破、寿命延长 |
| 复仇型 | 报仇雪恨、讨回公道 | 仇人伏诛、真相大白 |
| 守护型 | 保护宗门/家人/道侣 | 以弱胜强、守护成功 |
| 称霸型 | 称霸修仙界、一统天下 | 击败强敌、建立势力 |
| 探索型 | 探寻天道真相、上古秘密 | 真相揭晓、发现秘境 |

**追求的三个层次**：
```
表层追求：具体目标（突破金丹、击败某人、获得某物）
深层追求：身份认同（证明自己、被认可、超越前人）
终极追求：存在意义（证道成仙、守护苍生、逆天改命）
```

---

### 3. 手段（Means）── 他凭什么？

手段是角色实现追求的**能力支撑**，在玄幻修仙中尤为重要。

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 灵根资质 | 与生俱来的修炼天赋 | 五行灵根、变异灵根、特殊体质 |
| 功法传承 | 修炼的功法等级 | 凡级、灵级、玄级、天级功法 |
| 法宝神通 | 战斗手段 | 攻击法宝、防御法宝、辅助神通 |
| 机缘造化 | 特殊际遇 | 上古传承、秘境机缘、高人指点 |
| 性格优势 | 内在优势 | 冷静、隐忍、狠辣、机智 |

**玄幻修仙手段设计原则**：
1. **有限制**：无限制的能力=无聊（功法要有修炼条件/代价）
2. **有成长**：手段要能升级，给读者期待（境界提升、功法进阶）
3. **有特色**：避免同质化（独特的功法特性、罕见的灵根）

**境界体系示例**：
```
炼气期（1-9层）→ 筑基期 → 金丹期 → 元婴期 → 化神期 → 渡劫期 → 大乘期
```

## 二、玄幻修仙爽点定位设计

> 玄幻修仙角色的核心价值：**读者看他修炼、战斗、装逼爽不爽？**

### 爽点类型矩阵（玄幻修仙特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **装逼打脸** | 隐藏实力，关键时刻爆发 | 痛快、解气 | 扮猪吃虎、隐世高手 |
| **境界碾压** | 越级挑战，快速突破 | 兴奋、期待 | 妖孽天才、逆天资质 |
| **废柴逆袭** | 从废材到天才 | 共情、爽感 | 被退婚→打脸所有人 |
| **宝物收集** | 获得法宝、功法、丹药 | 满足感 | 机缘流、收集癖 |
| **宗门争锋** | 宗门大比、门派竞争 | 紧张→释放 | 宗门核心弟子 |
| **秘境探索** | 发现上古遗迹、获得传承 | 好奇→惊喜 | 探险流、寻宝流 |

### 主角爽点定位模板（玄幻修仙）

```
【爽点定位】
- 主爽点：境界碾压 + 装逼打脸
- 辅爽点：宝物收集
- 爽点触发条件：被轻视/挑衅时，或境界突破时
- 爽点表现形式：平时低调修炼，关键时刻展现实力震惊全场

【境界设定】
- 当前境界：筑基初期
- 真实战力：可越级战金丹中期
- 隐藏底牌：上古功法、特殊体质

【成长路线】
- 短期目标：突破金丹期
- 中期目标：成为宗门核心弟子
- 长期目标：证道成仙
```

## 五、检查清单

创建或审核角色时，必须确认：

### 主角必查项
- [ ] 底色是否清晰？（出身+经历→行事风格）
- [ ] 追求是否明确？（读者知道他要干什么）
- [ ] 灵根/资质是否有特色？（不是千篇一律的废材逆袭）
- [ ] 功法是否有特色？（不是千篇一律的系统）
- [ ] 境界体系是否清晰？（读者能理解实力层次）
- [ ] 爽点定位是否清晰？（读者看他干什么爽）
- [ ] 手段是否有限制？（避免无敌无聊）

### 配角必查项
- [ ] 是否符合重要度分级的设计深度？
- [ ] 境界是否合理？（不能随便越级）
- [ ] 是否有反工具人设计？
- [ ] 与主角的关系是否合理？
- [ ] 是否有辨识度（记忆点）？

### 全局检查
- [ ] 角色之间境界差距是否合理？
- [ ] 角色数量是否合理？（避免人物过多）
- [ ] 功法/法宝体系是否平衡？
- [ ] 是否避免了修仙题材的常见雷点？（如境界混乱、战力崩坏）

# 核心指令：玄幻修仙期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化修仙剧情的爽点节奏
*   读者反馈"不够爽"或"平淡"
*   规划境界突破前的铺垫
*   设计主角修炼成长曲线

---

## GAP 三角模型：修仙期待感引擎

> 期待感不是凭空产生的，而是由**差距**和**解决路径**共同构成的。

```
【差距 Gap】→ 境界差距/实力差距（读者："这也太弱了"）
     ↓
【期待 Anticipation】→ 功法传承/机缘出现（读者："有戏！能突破！"）
     ↓
【满足 Payoff】→ 境界突破/碾压敌人（读者："爽！终于突破了..."）
```

### 玄幻修仙 GAP 三要素

| 要素 | 作用 | 修仙设计要点 | 示例 |
|-----|------|-------------|------|
| 差距 Gap | 制造不满足感 | 境界差距、资质差距、资源差距 | 炼气期 vs 金丹期强敌 |
| 期待 Anticipation | 提供解决路径 | 获得传承、发现秘境、遇到高人 | 意外获得上古功法 |
| 满足 Payoff | 释放期待 | 境界突破、实力碾压、真相大白 | 突破金丹碾压敌人 |

---

## 玄幻修仙期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 成长期待 | 获得功法、遇到机缘 | 境界突破、实力飞跃 | 兴奋、期待 | 炼气→筑基突破 |
| 复仇期待 | 被欺压、宗门被灭 | 实力碾压、仇人伏诛 | 解气、痛快 | 回宗门碾压仇敌 |
| 探索期待 | 发现秘境、古籍线索 | 获得传承、解开谜团 | 好奇、震撼 | 进入上古遗迹 |
| 宝物期待 | 听闻神兵利器 | 获得法宝、炼化成功 | 满足、兴奋 | 获得先天灵宝 |
| 争锋期待 | 宗门大比、天骄争锋 | 力压群雄、一战成名 | 紧张→释放 | 宗门大比夺魁 |

### 多维期待叠加

同一场景可以同时触发多种期待：

```
示例：主角被宗门除名，流落荒野
→ 成长期待：在荒野中获得机缘突破
→ 复仇期待：回到宗门打脸所有人
→ 探索期待：发现上古遗迹
→ 宝物期待：获得上古传承
```

## 延迟满足法则（玄幻修仙）

### 黄金延迟公式

```
最佳延迟 = 期待强度 × 延迟系数

修仙延迟系数参考：
- 微期待（学新功法）：1-2章
- 中期待（小境界突破）：3-5章
- 大期待（大境界突破）：1-2卷
- 终极期待（证道飞升）：全书
```

### 修仙特有延迟技巧

| 技巧 | 说明 | 修仙示例 |
|-----|------|---------|
| 境界瓶颈 | 角色卡在某个境界 | 金丹期瓶颈，需要特殊机缘 |
| 资源不足 | 缺少突破所需资源 | 需要天材地宝炼制突破丹 |
| 强敌阻碍 | 有人阻止角色突破 | 宗门长老打压，不让参加大比 |
| 秘境时限 | 必须等待秘境开启 | 百年一开的上古秘境 |

## 黄金三章检查表（玄幻修仙）

### 第一章：建立差距 (Gap)
- [ ] 主角初始境界是否足够低？（废材/低阶修士）
- [ ] 差距是否能让读者产生共情？（被欺压/被退婚）
- [ ] 是否展示了修仙世界的实力等级？
- [ ] 是否有第一个冲突/挫折？

### 第二章：提供期待路径 (Anticipation)
- [ ] 是否出现"解决路径"的暗示？（传承/系统/高人）
- [ ] 金手指是否登场或有暗示？
- [ ] 主角是否展现出隐藏潜力？（特殊灵根/体质）
- [ ] 是否有新的悬念钩子？

### 第三章：首次满足 (Payoff)
- [ ] 是否有第一个情绪值 +2 以上的小高潮？
- [ ] 主角是否初步展现实力？（首次战斗胜利）
- [ ] 是否确立了故事的"爽感类型"？
- [ ] 是否埋下了新的期待种子？

# 核心指令：玄幻修仙爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 玄幻修仙爽点三级分类

### 小爽点（+1~+2）
**定义**：修炼小进展、小胜利

**典型场景**：
- 学会新功法/神通
- 击败同阶或低阶妖兽
- 获得灵石/丹药/灵草
- 小境界突破（如炼气3层→4层）
- 在宗门考核中表现出色
- 获得长老/师兄认可
- 成功炼制丹药/阵法

**频率建议**：每2章一次

**作用**：维持修炼快感，让读者感受到持续成长

---

### 中爽点（+3~+4）
**定义**：阶段性胜利、重要突破

**典型场景**：
- 大境界突破（如炼气→筑基）
- 获得高阶功法/法宝
- 击败越阶强敌
- 宗门大比夺魁
- 在宗门中地位大幅提升
- 获得核心弟子资格
- 成功进入秘境并获得传承
- 打脸看不起自己的天骄

**频率建议**：每8章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：重大转折、终极高潮

**典型场景**：
- 跨越式境界突破（如金丹→元婴）
- 获得上古传承/先天灵宝
- 击败高阶强敌（越两个大境界）
- 真实身份揭晓（隐藏天才/上古血脉）
- 回到宗门碾压所有仇敌
- 成为宗门掌门/一方霸主
- 渡劫成功飞升

**频率建议**：每30章一次（约每卷1次）

**作用**：全书记忆点，读者口碑传播点

## 爽点节奏配置（玄幻修仙推荐）

### 高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 2章 | 修炼进展频繁 |
| 中爽间隔 | 8章 | 配合境界突破 |
| 大爽间隔 | 30章 | 每卷一次大高潮 |

### 配置原则
1. **修炼不断**：最长不超过3章无修炼进展
2. **境界分层**：大境界突破要有足够铺垫
3. **战斗高潮**：每卷至少一场越级战斗
4. **避免连突破**：连续2章大境界突破会导致战力崩坏

# 核心指令：玄幻修仙世界观构建专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：

*   世界观设定出现矛盾或漏洞

## 一、修仙世界核心三要素

### 1. 灵气法则
修仙世界的基础运行规则。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 灵气浓度 | 世界灵气的分布 | 灵气稀薄区→灵气充裕区→灵脉 |
| 灵气类型 | 五行灵气、特殊灵气 | 不同区域灵气属性不同 |
| 灵气枯竭 | 灵气是否会耗尽 | 制造资源争夺的核心矛盾 |

### 2. 境界体系
修仙者的实力分层。

**标准境界体系**：
```
凡人 → 炼气期(1-9层) → 筑基期(初/中/后) → 金丹期 → 元婴期 → 化神期 → 渡劫期 → 大乘期 → 飞升
```

**境界设计原则**：
1. **层次清晰**：每个境界的实力差距要明确
2. **突破条件**：每个境界的突破条件不同
3. **寿命对应**：境界越高寿命越长
4. **战力倍率**：高一个大境界约10倍战力差

**境界设计模板**：
```
【境界名称】
- 层次划分：初期/中期/后期/巅峰
- 标志特征：（如金丹期标志是凝结金丹）
- 寿命上限：
- 战力范围：
- 突破条件：
- 突破难度：
- 该境界的社会地位：
```

### 3. 天道规则
修仙世界的最高法则。

| 规则 | 说明 | 剧情作用 |
|-----|------|---------|
| 天劫 | 突破高阶境界需渡天劫 | 制造紧张感和仪式感 |
| 因果 | 杀人有因果报应 | 限制主角无脑杀戮 |
| 天道压制 | 高阶修士不能随意出手 | 保护低阶修士的生存空间 |
| 气运 | 主角有特殊气运 | 解释主角的机缘 |

## 二、宗门势力体系

### 势力等级

| 等级 | 最强修士 | 规模 | 地位 |
|-----|---------|------|------|
| 超级宗门 | 大乘期 | 万人以上 | 一方霸主 |
| 一流宗门 | 化神期 | 千人以上 | 区域强者 |
| 二流宗门 | 元婴期 | 百人以上 | 地方势力 |
| 三流宗门 | 金丹期 | 数十人 | 小门小派 |
| 散修联盟 | 不定 | 不定 | 松散组织 |

### 宗门内部结构

```
宗主
  ├─ 太上长老（闭关修炼）
  ├─ 大长老（日常管理）
  ├─ 各峰峰主
  │    ├─ 内门弟子
  │    └─ 外门弟子
  └─ 执法堂/丹药堂/器物堂
```

### 势力关系设计

**核心矛盾来源**：
1. **资源争夺**：灵脉、矿脉、灵草产地
2. **领地纷争**：势力范围的扩张和防守
3. **正邪对立**：正道宗门 vs 魔道势力
4. **内部斗争**：宗门内部的派系之争

---

## 三、功法与法宝体系

### 功法等级

| 等级 | 稀有度 | 适用境界 | 获取难度 |
|-----|-------|---------|---------|
| 凡级 | 常见 | 炼气期 | 宗门发放 |
| 灵级 | 较少 | 筑基-金丹 | 宗门核心 |
| 玄级 | 稀有 | 元婴-化神 | 秘境/传承 |
| 天级 | 极稀有 | 渡劫-大乘 | 上古遗迹 |
| 仙级 | 传说 | 飞升 | 仙人遗留 |

### 法宝等级

| 等级 | 威力 | 获取方式 |
|-----|------|---------|
| 法器 | 辅助修炼 | 自行炼制 |
| 灵器 | 增强战力 | 宗门奖励 |
| 宝器 | 大幅增强 | 秘境获取 |
| 灵宝 | 改变战局 | 上古传承 |
| 仙器 | 毁天灭地 | 仙人遗留 |

## 四、世界观设计检查清单

- [ ] 灵气法则是否自洽？
- [ ] 境界体系是否层次清晰？
- [ ] 每个境界的突破条件是否明确？
- [ ] 宗门势力格局是否合理？
- [ ] 功法/法宝等级是否与境界匹配？
- [ ] 天道规则是否能限制战力崩坏？
- [ ] 是否有足够的资源争夺点（制造冲突）？
- [ ] 世界观是否为剧情服务（而非为设定而设定）？
