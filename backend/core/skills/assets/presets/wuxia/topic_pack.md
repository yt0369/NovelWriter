# 核心指令：武侠江湖角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 武侠江湖示例 |
|-----|------|-------------|
| 出身 | 门派、家世、师承 | 名门正派弟子、江湖孤儿、隐世高人之徒 |
| 核心经历 | 塑造性格的关键事件 | 师门被灭、被逐出师门、意外获得武学秘籍 |
| 创伤/执念 | 促成行为模式的心理锚点 | "为师门报仇"、"成为天下第一"、"行侠仗义" |

**底色公式**：
```
底色 = 师承出身 + 江湖经历 → 形成的侠义观/行事风格
```

**武侠江湖示例**：
- 主角底色：少林俗家弟子 + 师父被害、被逐出师门 → "外表憨厚、内心坚韧、不畏强权"
- 反派底色：魔教教主之子 + 从小被灌输仇恨 → "手段狠辣、内心矛盾、渴望被认可"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 复仇型 | 为师门/家人报仇 | 仇人伏诛、真相大白 |
| 称霸型 | 成为天下第一 | 击败强敌、武林盟主 |
| 侠义型 | 行侠仗义、锄强扶弱 | 以弱胜强、守护成功 |
| 探索型 | 寻找绝世武学 | 获得秘籍、武功大成 |
| 自由型 | 逍遥江湖、不受束缚 | 快意恩仇、潇洒自在 |

**追求的三个层次**：
```
表层追求：具体目标（击败某人、找到某本秘籍）
深层追求：身份认同（证明自己的武学、被江湖认可）
终极追求：存在意义（守护武林正义、追求武道巅峰）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 武功修为 | 内力深厚程度 | 内力等级、真气类型 |
| 武学招式 | 习得的武功招式 | 拳法、剑法、掌法、暗器 |
| 轻功身法 | 移动和闪避能力 | 凌波微步、梯云纵 |
| 江湖人脉 | 可调用的关系 | 门派关系、江湖朋友、暗中势力 |
| 侠义之心 | 内在品质 | 正义感、义气、不屈不挠 |

**武侠手段设计原则**：
1. **有传承**：武功要有来历（师承、秘籍、机缘）
2. **有克制**：不同武功之间有相克关系
3. **有境界**：武功修炼有明确的境界划分
4. **有代价**：强大的武功有修炼条件或副作用

**武功境界体系**：
```
初窥门径 → 登堂入室 → 炉火纯青 → 出神入化 → 返璞归真
```

## 三、配角分层设计（武侠江湖）

### 角色重要度分级

| 级别 | 角色 | 设计深度 | 核心要素 |
|-----|------|---------|---------|
| S级 | 主角 | 完整三维+武功体系 | 底色+追求+手段+武功+爽点定位 |
| A级 | 核心配角 | 完整三维 | 底色+追求+手段+武功+与主角恩怨 |
| B级 | 重要配角 | 简化三维 | 一个追求+门派+武功+与主角关系 |
| C级 | 功能配角 | 单点设计 | 功能定位+武功+一个记忆点 |
| D级 | 背景板 | 无需设计 | 遵守主体性原则即可 |

### 武侠特有角色类型

| 类型 | 作用 | 设计要点 |
|-----|------|---------|
| 师父 | 传授武功、引导成长 | 要有自己的故事和遗憾 |
| 宿敌 | 长期对手、互相成就 | 实力相当、亦敌亦友 |
| 红颜知己 | 情感线、动机补充 | 有自己的武功和追求 |
| 江湖前辈 | 提供信息、指点迷津 | 有自己的江湖往事 |

## GAP 三角模型：武侠江湖期待感引擎

```
【差距 Gap】→ 武功差距/恩怨未了（读者："他打不过啊"）

【期待 Anticipation】→ 获得秘籍/苦练武功（读者："有戏！要变强了！"）
     ↓
【满足 Payoff】→ 武功大成/快意恩仇（读者："爽！终于报仇了..."）
```

## 武侠江湖爽点三级分类

### 小爽点（+1~+2）
**定义**：武功小进展、小胜利

**典型场景**：
- 学会新招式
- 内力小有精进
- 击败普通对手
- 获得江湖朋友的认可
- 路见不平出手相助
- 发现武学线索
- 在比武中展现实力

**频率建议**：每4章一次

**作用**：维持江湖氛围，让读者感受到武功成长

---

### 中爽点（+3~+4）
**定义**：武功突破、重要胜利

**典型场景**：
- 武功境界突破（如登堂入室→炉火纯青）
- 获得武学秘籍
- 击败强敌
- 门派大比获胜
- 了结一段恩怨
- 获得江湖名号
- 救下重要人物

**频率建议**：每15章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：武功大成、终极对决

**典型场景**：
- 领悟绝世武学
- 击败宿敌/终极对决
- 武林大会夺魁
- 成为一代大侠
- 终极复仇成功
- 天下第一

**频率建议**：每50章一次

**作用**：全书记忆点，读者口碑传播点

---

## 爽点节奏配置（武侠江湖推荐）

### 中低频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 4章 | 武功进展和小战斗 |
| 中爽间隔 | 15章 | 武功突破和重要胜利 |
| 大爽间隔 | 50章 | 终极对决和武功大成 |

### 配置原则
1. **武功递进**：武功成长要有明确的阶梯感
2. **恩怨分明**：恩怨了结要有足够铺垫
3. **战斗精彩**：每场战斗都要有看点（招式描写、策略运用）
4. **侠义贯穿**：侠义精神要贯穿始终

### 1. 武功体系

**武功等级**：
| 等级 | 稀有度 | 威力 | 获取方式 |
|-----|-------|------|---------|
| 三流武功 | 常见 | 普通 | 门派基础 |
| 二流武功 | 较少 | 较强 | 门派核心 |
| 一流武功 | 稀有 | 强大 | 掌门传承 |
| 绝世武功 | 极稀有 | 绝顶 | 秘籍/机缘 |

**武功修炼境界**：
```
初窥门径（刚入门）→ 登堂入室（小有所成）→ 炉火纯青（精通）
→ 出神入化（大师）→ 返璞归真（宗师）
```

**内力体系**：
- 内力深浅决定基础战力
- 不同门派内力有不同特性（少林刚猛、武当绵柔）
- 内力可以通过修炼、服药、传功提升

### 2. 门派体系

**门派等级**：
| 等级 | 掌门实力 | 弟子数量 | 江湖地位 |
|-----|---------|---------|---------|
| 超级门派 | 返璞归真 | 千人以上 | 武林泰斗 |
| 一流门派 | 出神入化 | 数百人 | 江湖名门 |
| 二流门派 | 炉火纯青 | 数十人 | 地方势力 |
| 三流门派 | 登堂入室 | 十余人 | 小门小派 |

**门派内部结构**：
```
掌门
  ├─ 太上长老（退隐前辈）
  ├─ 长老会
  ├─ 各堂/各峰
  │    ├─ 入室弟子
  │    └─ 记名弟子
  └─ 外门弟子
```

### 3. 江湖规矩

| 规矩 | 说明 | 剧情作用 |
|-----|------|---------|
| 武林盟约 | 正道门派的联盟约定 | 制造正邪对立 |
| 比武规矩 | 点到为止、不伤性命 | 限制无脑杀戮 |
| 师徒之义 | 一日为师终身为父 | 制造情感冲突 |
| 江湖道义 | 路见不平拔刀相助 | 驱动侠义行为 |
| 恩怨分明 | 有恩必报有仇必报 | 驱动复仇线 |

## 二、势力格局

### 正邪对立

```
正道联盟
  ├─ 少林（武学正宗）
  ├─ 武当（内功深厚）
  ├─ 峨眉（女子门派）
  └─ 其他正道门派

邪道势力
  ├─ 魔教（最大邪派）
  ├─ 血衣楼（杀手组织）
  └─ 其他邪道势力

中立势力
  ├─ 丐帮（天下第一大帮）
  ├─ 镖局（商业组织）
  └─ 隐世门派
```

### 势力冲突来源
1. **正邪对立**：正道vs魔道的根本矛盾
2. **武林至尊**：争夺武林盟主之位
3. **绝世秘籍**：争夺失传的武学秘籍
4. **领地纷争**：门派势力范围的争夺
5. **个人恩怨**：门派之间的历史恩怨
