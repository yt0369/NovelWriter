# 核心指令：无限流角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 无限流示例 |
|-----|------|-------------|
| 出身 | 背景、起点 | 普通人、退伍军人、医生、学生、罪犯 |
| 核心经历 | 塑造性格的关键事件 | 首次副本团灭、队友背叛、亲人死亡、被主神选中 |
| 创伤/执念 | 促成行为模式的心理锚点 | "活下去"、"揭开主神秘密"、"拯救队友" |

**底色公式**：
```
底色 = 进入主神空间的原因 + 首次副本经历 → 形成的生存态度
```

**无限流示例**：
- 主角底色：普通学生 + 首次副本险些团灭 → "谨慎冷静、重视情报"
- 队友底色：退伍军人 + 多次副本生还 → "果断狠辣、相信实力"

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 生存型 | 活下去、回到现实 | 通关副本、避免死亡 |
| 探索型 | 揭开主神秘密 | 发现真相、解开谜团 |
| 强大型 | 成为最强轮回者 | 获得强力能力、击败强敌 |
| 拯救型 | 拯救队友/世界 | 救援成功、改变命运 |
| 复仇型 | 向主神/背叛者复仇 | 复仇成功、击败仇人 |

**追求的三个层次**：
```
表层追求：具体目标（通关某个副本、获得某个能力）
深层追求：身份认同（证明自己能活下去、被认可为强者）
终极追求：存在意义（揭开主神秘密、打破轮回、拯救所有人）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 能力多样性 | 多种能力组合 | 异能、武器、道具、技能 |
| 适应力 | 快速适应副本 | 情报分析、规则理解、应变能力 |
| 团队协作 | 队友配合 | 分工合作、互补短板、信任默契 |
| 副本经验 | 熟悉副本规则 | 剧情预知、机制理解、套路应对 |
| 资源管理 | 积分/道具管理 | 合理分配、投资回报、长远规划 |

**无限流手段设计原则**：
1. **有短板**：不能所有方面都最强（战斗强但智力弱）
2. **有成长**：能力要能提升（从萌新到资深轮回者）
3. **有特色**：要有标志性的能力（独特的能力组合）
4. **有对手**：对手也要有特色和实力

---

## 二、无限流爽点定位设计

> 无限流角色的核心价值：**读者看他通关副本、获得能力、揭开秘密爽不爽？**

### 爽点类型矩阵（无限流特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **副本通关** | 惊险通关、完美通关 | 紧张→释放 | 副本之王、通关专家 |

| **团队配合** | 完美的团队协作 | 热血→感动 | 队长、团队核心 |

当你发现以下情况时，**主动激活此技能**：

## GAP 三角模型：无限流期待感引擎

```
【差距 Gap】→ 副本难度/能力差距（读者："能通关吗？"）
     ↓
【期待 Anticipation】→ 能力准备/情报收集（读者："有戏！准备好了！"）
     ↓
【满足 Payoff】→ 副本通关/能力获得（读者："爽！通关了！能力到手！"）
```

### 无限流期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 挑战期待 | 即将进入高难副本 | 副本通关 | 紧张→释放 | 通关恐怖副本、完成主线任务 |
| 成长期待 | 积分积累/能力瓶颈 | 获得新能力 | 期待→兴奋 | 兑换强力能力、能力进化 |
| 悬念期待 | 发现主神线索 | 秘密揭开 | 好奇→震撼 | 揭开主神真相、发现隐藏剧情 |
| 团队期待 | 队友危机 | 队友救援 | 担心→感动 | 救回队友、团队重聚 |
| 复仇期待 | 遇到仇人/背叛者 | 复仇成功 | 解气→痛快 | 击败背叛者、报仇雪恨 |

---

## 满足感分级详解（无限流）

| 等级 | 情绪值 | 触发频率 | 典型形式 | 无限流示例 |
|-----|-------|---------|---------|-----------|
| 微爽 | +1 | 每1章 | 小进展 | 完成支线任务、获得道具、击败小 Boss |

## 无限流爽点三级分类

### 小爽点（+1~+2）
**定义**：支线任务完成、小 Boss 击败

**典型场景**：
- 完成支线任务
- 获得道具/装备
- 击败小 Boss
- 发现隐藏剧情
- 获得队友认可
- 通关普通副本
- 获得基础能力
- 积分增加

**频率建议**：每1章一次

**作用**：维持紧张感，让读者感受到持续进展

---

### 中爽点（+3~+4）
**定义**：困难副本通关、强力能力获得

**典型场景**：
- 通关困难副本
- 获得强力能力
- 击败强敌
- 救援队友
- 发现重要线索
- 能力进化
- 完成主线任务
- 团队配合成功

**频率建议**：每5章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：终极副本通关、主神秘密揭开

**典型场景**：
- 通关终极副本
- 揭开主神秘密
- 成为最强轮回者
- 打破轮回
- 成为主神
- 拯救所有人
- 改变主神空间规则

**频率建议**：每20章一次

**作用**：全书记忆点，读者口碑传播点

## 爽点节奏配置（无限流推荐）

### 高频节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 每章都有进展/任务完成 |
| 中爽间隔 | 5章 | 频繁的副本通关/能力获得 |
| 大爽间隔 | 20章 | 重大秘密揭开 |

### 配置原则
1. **紧张不断**：最长不超过2章无危机/任务
2. **副本递进**：副本难度要逐步提升
3. **能力成长**：能力要有可见的提升
4. **秘密揭开**：主神秘密要逐步揭开

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 选人机制 | 如何选中轮回者 | 随机/特定条件/主动选择 |
| 任务机制 | 如何分配任务 | 主线/支线/隐藏任务 |
| 惩罚机制 | 失败的后果 | 抹杀/扣分/降级 |
| 奖励机制 | 成功的奖励 | 积分/能力/道具 |

**主神规则设计原则**：
1. **公平性**：规则要对所有人公平
2. **挑战性**：任务要有足够难度
3. **成长性**：要有明确的成长路径
4. **悬念性**：要有未解之谜

### 2. 副本规则

**副本类型**：
- 电影副本：基于电影世界
- 动漫副本：基于动漫世界
- 游戏副本：基于游戏世界
- 小说副本：基于小说世界
- 原创副本：完全原创世界

**难度分级**：
```
新手副本（难度1-3）→ 普通副本（难度4-6）→ 困难副本（难度7-9）
→ 恐怖副本（难度10-12）→ 终极副本（难度13-15）
```

**任务机制**：
| 任务类型 | 说明 | 奖励 | 惩罚 |
|---------|------|------|------|
| 主线任务 | 必须完成 | 高 | 抹杀 |
| 支线任务 | 可选完成 | 中 | 无 |
| 隐藏任务 | 需要发现 | 极高 | 无 |
| 团队任务 | 团队协作 | 高 | 扣分 |

### 3. 时间规则

**副本时间**：
- 现实时间：副本内时间与现实同步
- 加速时间：副本内时间加速（1天=1年）
- 暂停时间：现实时间暂停

**休息时间**：
- 副本间隔：每次副本后的休息时间
- 修整期：可以在主神空间修整
- 自由活动：可以进入其他副本世界

## 二、能力体系

### 能力分类

| 类型 | 说明 | 示例 |
|-----|------|------|
| 异能类 | 超能力 | 念动力、火焰、冰冻 |
| 武器类 | 武器精通 | 剑术、枪术、弓术 |
| 魔法类 | 魔法能力 | 元素魔法、召唤魔法 |
| 科技类 | 科技装备 | 机甲、纳米装甲 |
| 血统类 | 血脉能力 | 吸血鬼、狼人、龙族 |
| 技能类 | 特殊技能 | 医疗、黑客、侦查 |

### 能力等级

| 等级 | 威力 | 积分消耗 | 获取难度 |
|-----|------|---------|---------|
| D级 | 低 | 100-500 | 新手副本 |
| C级 | 中 | 500-2000 | 普通副本 |
| B级 | 高 | 2000-5000 | 困难副本 |
| A级 | 极高 | 5000-10000 | 恐怖副本 |
| S级 | 毁灭性 | 10000+ | 终极副本 |

### 能力组合

**组合原则**：
1. **互补性**：能力要互补短板
2. **协同性**：能力要能协同作战
3. **成长性**：能力要能升级进化
4. **限制性**：能力要有使用限制

**组合示例**：
```
近战能力 + 防御能力 + 恢复能力 = 坦克型
远程能力 + 爆发能力 + 机动能力 = 输出型
辅助能力 + 治疗能力 + 侦查能力 = 辅助型
```

## 三、积分系统

### 积分获取

| 来源 | 说明 | 积分量 |
|-----|------|-------|
| 主线任务 | 完成主线 | 1000-5000 |
| 支线任务 | 完成支线 | 100-1000 |
| 隐藏任务 | 发现隐藏 | 2000-10000 |
| 击杀奖励 | 击杀敌人 | 10-500 |
| 评价奖励 | 通关评价 | 0-5000 |

### 积分消耗

| 用途 | 说明 | 积分量 |
|-----|------|-------|
| 能力兑换 | 兑换新能力 | 100-10000+ |
| 能力升级 | 升级现有能力 | 500-5000 |
| 道具购买 | 购买道具 | 10-1000 |
| 属性强化 | 强化身体属性 | 100-1000 |
| 复活队友 | 复活死亡队友 | 5000-20000 |

### 积分管理

**投资策略**：
- 前期：优先基础能力和生存能力
- 中期：强化主要能力和团队配合
- 后期：追求顶级能力和能力组合
