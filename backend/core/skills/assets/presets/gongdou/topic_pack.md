# 核心指令：宫斗权谋角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他/她从哪里来？

| 维度 | 说明 | 宫斗权谋示例 |
|-----|------|-------------|
| 出身 | 家世、派系、地位 | 没落世家嫡女、权臣庶女、寒门才女 |
| 核心经历 | 塑造性格的关键事件 | 被陷害入冷宫、亲眼目睹母亲被害、家族被灭 |
| 创伤/执念 | 促成行为模式的心理锚点 | "绝不再任人摆布"、"要让害我的人付出代价" |

**底色公式**：
```
底色 = 出身（家世+派系） + 核心经历 → 形成的行事风格
```

**宫斗权谋示例**：
- 主角底色：没落世家嫡女 + 被继母陷害入冷宫 → "表面温顺、内心狠辣"的行事风格
- 反派底色：权臣之女 + 从小被教导权力至上 → "不择手段、视人命如草芥"的人格

---

### 2. 追求（Drive）── 他/她要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 复仇型 | 为家人/自己复仇 | 仇人伏诛、真相大白 |
| 上位型 | 追求权力巅峰 | 地位提升、掌控朝堂 |
| 守护型 | 保护家人/爱人 | 化解阴谋、守护成功 |
| 真相型 | 揭开隐藏的真相 | 真相大白、阴谋揭露 |
| 自由型 | 摆脱权力斗争 | 脱离牢笼、获得自由 |

**追求的三个层次**：
```
表层追求：具体目标（扳倒某人、获得某个位置）
深层追求：身份认同（证明自己的价值、不再被人操控）
终极追求：存在意义（守护家族、改变制度、获得真正的自由）
```

---

### 3. 手段（Means）── 他/她凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 智谋 | 计谋和策略 | 借刀杀人、以退为进、离间计 |
| 人脉 | 可调用的关系网 | 宫中眼线、朝堂盟友、暗中势力 |
| 信息 | 掌握的秘密 | 他人把柄、宫中秘闻、隐藏身世 |
| 才艺 | 个人魅力和才能 | 琴棋书画、医术、辩才 |
| 隐忍 | 内在优势 | 忍辱负重、伺机而动、不露声色 |

**宫斗权谋手段设计原则**：
1. **有代价**：每次出手都有风险（暴露、被反算计）
2. **有层次**：计谋要有明暗两层（表面一套、暗中一套）
3. **有对手**：对手也要有智谋（不能只有主角聪明）
4. **有底线**：角色的底线决定了他/她的行为边界

## 二、宫斗权谋爽点定位设计

> 宫斗权谋角色的核心价值：**读者看他/她算计、反转、上位爽不爽？**

### 爽点类型矩阵（宫斗权谋特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **以退为进** | 示弱诱敌，反手致命一击 | 惊喜、过瘾 | 隐忍型、扮猪吃虎 |
| **借刀杀人** | 利用他人之手除掉对手 | 佩服、痛快 | 智谋型、幕后操控 |
| **真相反转** | 揭露隐藏的真相 | 震撼、解气 | 真相追寻者 |
| **绝地反击** | 在绝境中翻盘 | 紧张→释放 | 不屈型、逆境翻盘 |
| **步步为营** | 精密布局逐步上位 | 满足感 | 棋手型、长线布局 |
| **人心收服** | 收服人心、建立势力 | 成就感 | 领袖型、人心所向 |

### 主角爽点定位模板（宫斗权谋）

```
【爽点定位】
- 主爽点：以退为进 + 真相反转
- 辅爽点：步步为营
- 爽点触发条件：被陷害/被逼入绝境时
- 爽点表现形式：表面落入圈套，实则早已布局，反手将对手一军

【权谋设定】
- 当前地位：[低位/中位/高位]
- 核心盟友：
- 主要对手：
- 掌握的秘密：
- 隐藏的底牌：

【上位路线】
- 短期目标：在宫中站稳脚跟
- 中期目标：扳倒主要对手
- 长期目标：掌控权力核心
```

## 三、配角分层设计（宫斗权谋）

### 角色重要度分级

| 级别 | 角色 | 设计深度 | 核心要素 |
|-----|------|---------|---------|
| S级 | 主角 | 完整三维+权谋设定 | 底色+追求+手段+派系+爽点定位 |
| A级 | 核心配角 | 完整三维 | 底色+追求+手段+派系+与主角关系 |
| B级 | 重要配角 | 简化三维 | 一个追求+派系+特点+与主角关系 |
| C级 | 功能配角 | 单点设计 | 功能定位+派系+一个记忆点 |
| D级 | 背景板 | 无需设计 | 遵守主体性原则即可 |

### 宫斗角色关系设计要点

**每个角色都要有"三面"**：
1. **表面立场**：公开表现出的态度
2. **真实立场**：内心真正的想法
3. **隐藏动机**：不为人知的目的

**角色关系矩阵**：
```
         主角  反派A  反派B  盟友A  盟友B
主角      -    敌对   暗敌   明友   暗友
反派A    敌对   -     暗盟   敌对   利用
反派B    暗敌  暗盟    -     中立   敌对
盟友A    明友  敌对   中立    -     竞争
盟友B    暗友  利用   敌对   竞争    -
```

# 核心指令：宫斗权谋期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化宫斗剧情的爽点节奏

*   规划重大反转前的铺垫
*   设计主角上位曲线

## GAP 三角模型：宫斗权谋期待感引擎

```
【差距 Gap】→ 地位差距/权力差距（读者："她也太惨了"）
     ↓
【期待 Anticipation】→ 发现秘密/布局开始（读者："有戏！要反击了！"）
     ↓
【满足 Payoff】→ 计谋成功/真相大白（读者："爽！终于扳倒了..."）
```

### 宫斗权谋 GAP 三要素

| 要素 | 作用 | 宫斗设计要点 | 示例 |
|-----|------|-------------|------|
| 差距 Gap | 制造不满足感 | 地位差距、权力差距、信息差距 | 冷宫妃子 vs 宠妃 |
| 期待 Anticipation | 提供解决路径 | 发现秘密、获得盟友、布局开始 | 发现宠妃的把柄 |
| 满足 Payoff | 释放期待 | 计谋成功、真相大白、对手落败 | 宠妃被废、真相揭露 |

---

## 宫斗权谋期待感五维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 复仇期待 | 被陷害、家人被害 | 仇人伏诛、真相大白 | 快意、解恨 | 揭露害母真凶 |
| 上位期待 | 地位低下、被压制 | 地位提升、掌权 | 满足、成就 | 从冷宫到皇后 |
| 悬念期待 | 疑点、阴谋线索 | 真相揭晓、阴谋败露 | 好奇、震撼 | 揭开宫中大秘密 |
| 反转期待 | 主角陷入困境 | 绝地反击、反败为胜 | 紧张→释放 | 被陷害后反手一击 |
| 情感期待 | 暧昧、误会、阻碍 | 真心相待、误会解除 | 感动、心动 | 帝王真心相待 |

### 多维期待叠加

```
示例：主角被诬陷下毒，打入冷宫
→ 复仇期待：找出真凶，让她付出代价
→ 上位期待：从冷宫重新崛起
→ 悬念期待：下毒案背后的更大阴谋
→ 反转期待：利用冷宫的机会暗中布局
```

# 核心指令：宫斗权谋爽点节奏管理专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
* 项目初始化时设定爽点节奏
* 章节规划时分配爽点等级
* 审核剧情时检查爽点密度
* 发现爽点分布不均

---

## 宫斗权谋爽点三级分类

### 小爽点（+1~+2）
**定义**：小计谋成功、小线索发现

**典型场景**：
- 发现对手的一个弱点
- 成功收买一个眼线
- 在言语交锋中占上风
- 化解一次小陷害
- 获得一个有用的情报
- 成功离间对手的一个盟友
- 在宴会上不动声色地反击

**频率建议**：每5章一次

**作用**：维持权谋氛围，让读者感受到暗流涌动

---

### 中爽点（+3~+4）
**定义**：重要计谋成功、地位提升

**典型场景**：
- 成功扳倒一个对手
- 地位大幅提升（如从嫔升为妃）
- 揭露一个重要真相
- 获得帝王/权臣的信任
- 成功化解一次重大危机
- 收服一个重要盟友
- 对手遭受重大打击

**频率建议**：每15章一次

**作用**：形成节奏高潮，给读者阶段性满足

---

### 大爽点（+5）
**定义**：重大权力变动、终极真相

**典型场景**：
- 扳倒主要对手（宠妃被废/权臣落马）
- 揭露宫中最大的秘密
- 登上权力巅峰（成为皇后/摄政）
- 终极复仇成功
- 所有阴谋真相大白
- 多年布局一朝收网

**频率建议**：每50章一次

**作用**：全书记忆点，读者口碑传播点

---

## 爽点节奏配置（宫斗权谋推荐）

### 低频大爽节奏

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 5章 | 暗流涌动，小计谋频出 |
| 中爽间隔 | 15章 | 重要计谋成功 |
| 大爽间隔 | 50章 | 重大权力变动 |

### 配置原则
1. **铺垫充分**：大爽点前至少有10章铺垫
2. **暗线不断**：即使没有明面爽点，暗线也要推进
3. **反转质量**：宁可少一个反转，也要保证每个反转的质量
4. **避免弱智反派**：对手的计谋也要有水平

当你发现以下情况时，**主动激活此技能**：

## 一、宫斗世界观核心三要素

### 1. 后宫等级体系
后宫的层级结构和晋升路线。

| 等级 | 地位 | 权力范围 | 住所 | 侍女人数 |
|-----|------|---------|------|---------|
| 皇后 | 正宫 | 统管六宫 | 坤宁宫 | 20+ |
| 皇贵妃 | 副后 | 协管六宫 | 承乾宫 | 15 |
| 贵妃 | 高位 | 独立管理 | 永和宫 | 12 |
| 妃 | 中高位 | 部分管理 | 各宫主位 | 8 |
| 嫔 | 中位 | 自管一宫 | 各宫侧位 | 6 |
| 贵人 | 中低位 | 无管理权 | 合住 | 4 |
| 常在 | 低位 | 无管理权 | 合住 | 2 |
| 答应 | 最底层 | 无管理权 | 合住 | 1 |

**晋升规则**：
1. **家世加成**：出身名门初始位份更高
2. **帝宠关键**：得宠是晋升的主要途径
3. **子嗣加分**：生育皇嗣可晋升
4. **功绩特殊**：立功可破格晋升
5. **太后影响**：太后的态度影响晋升

### 2. 前朝势力格局
朝廷权力结构对后宫的影响。

```
【前朝势力图】
皇帝
├── 外戚集团（太后/皇后家族）
├── 世家集团（百年大族）
├── 寒门新贵（科举出身）
├── 宦官势力（太监总管）
└── 宗室势力（皇亲国戚）
```

**前朝与后宫的联动**：
- 前朝势力支持后宫中的对应妃嫔
- 后宫的荣辱直接影响前朝家族地位
- 联姻是前朝与后宫的纽带

### 3. 派系关系网络

| 关系类型 | 特征 | 稳定性 | 典型模式 |
|---------|------|-------|---------|
| 利益联盟 | 基于共同利益 | 不稳定 | 联手对付共同的敌人 |
| 血缘联盟 | 基于家族关系 | 较稳定 | 姐妹/姑侄共同进退 |
| 依附关系 | 弱者依附强者 | 看实力 | 低阶妃嫔投靠高阶 |
| 暗中同盟 | 表面敌对暗中合作 | 最不稳定 | 明争暗斗中互相利用 |
| 主仆关系 | 主子与心腹 | 最稳定 | 女主与贴身侍女 |

### 后宫核心资源

## 四、世界观设计检查清单

- [ ] 后宫等级体系是否完整且清晰？
- [ ] 前朝势力格局是否与后宫联动？
- [ ] 派系关系是否复杂但有逻辑？
- [ ] 权力资源是否平衡？（不能一方独大）
- [ ] 角色晋升路线是否合理？（不能太快也不能太慢）
- [ ] 是否有足够的权力斗争空间？（派系、利益、人心）
- [ ] 历史细节是否到位？（服饰、礼仪、称谓、规矩）
- [ ] 世界观是否为剧情服务（而非为设定而设定）？
