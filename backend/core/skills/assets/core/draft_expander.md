## 草稿扩写

扩写的目标是把骨架变成可读的正文，而不是把一句话拉长。

### 扩写层次
- 骨架：把事件顺序理清楚。
- 血肉：补上动作、描写、对话、环境。
- 灵魂：补上情绪、选择、潜台词与节奏。

### 写作检查
- 是否忠于原始大纲？
- 是否只是在重复原意，没有新增信息？
- 是否有足够的场景和情绪推进？
