# 核心指令：男频角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

> ⚠️ **弃用旧模型**："欲望-恐惧-谎言"是好莱坞戏剧模型，不适合男频网文。
> 男频主角的核心是**追求**而非**克服**，是**变强**而非**治愈**。

### 1. 底色（Origin）── 他从哪里来？

底色决定角色的**行事风格**和**价值观**，是角色一切行为的底层逻辑。

| 维度 | 说明 | 示例 |
|-----|------|------|
| 出身 | 家世、阶层、种族 | 贫民窟孤儿、世家嫡子、异族混血 |
| 核心经历 | 塑造性格的关键事件 | 被灭门、被背叛、被压制十年 |
| 创伤/执念 | 促成行为模式的心理锚点 | "绝不再被人踩在脚下" |

**底色公式**：
```
底色 = 出身 + 核心经历 → 形成的执念/行事风格
```

**示例**：
- 主角底色：落魄世家庶子 + 被嫡系打压十年 → "隐忍发育、一击必杀"的行事风格
- 反派底色：暴发户新贵 + 曾被贵族羞辱 → "极度自卑转化的傲慢"人格

---

### 2. 追求（Drive）── 他要去哪里？

追求是角色的**行动引擎**，男频角色的追求通常比传统文学更**外在**、更**具体**。

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 生存型 | 活下去、保护家人 | 绝境翻盘、守护成功 |
| 复仇型 | 报仇、讨回公道 | 仇人伏诛、真相大白 |
| 野心型 | 称霸、成神、登顶 | 一统天下、站在巅峰 |
| 探索型 | 解谜、寻找真相 | 真相揭晓、发现新世界 |
| 守护型 | 保护某人/某地 | 以弱胜强、守护成功 |

**追求的三个层次**：
```
表层追求：具体目标（打败某人、获得某物）
深层追求：身份认同（证明自己、被认可）
终极追求：存在意义（成为什么样的存在）
```

---

### 3. 手段（Means）── 他凭什么？

手段是角色实现追求的**能力支撑**，是爽点的**技术来源**。

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 金手指 | 外挂式能力 | 系统、重生、穿越者知识 |
| 天赋 | 与生俱来的优势 | 特殊血脉、绝世资质 |
| 资源 | 可调用的人/物 | 贵人相助、神秘传承、家族势力 |
| 性格 | 内在优势 | 冷静、隐忍、狠辣、机智 |
| 技能 | 习得的能力 | 独门功法、特殊技艺 |

**手段设计原则**：
1. **有限制**：无限制的能力=无聊（系统要有冷却/代价）
2. **有成长**：手段要能升级，给读者期待
3. **有特色**：避免同质化（都是"系统"，功能要有区分）

## 二、爽点定位设计

> 男频角色的核心价值：**读者看他干什么爽？**

### 爽点类型矩阵

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **装逼打脸** | 低调装逼，关键时刻爆发 | 痛快、解气 | 隐世高手、实力深藏 |
| **扮猪吃虎** | 示弱诱敌，反手碾压 | 惊喜、过瘾 | 废柴逆袭、扮弱实强 |
| **碾压升级** | 越级挑战，快速成长 | 兴奋、期待 | 妖孽天才、系统拥有者 |
| **废柴逆袭** | 从低谷一路向上 | 共情、爽感 | 被看不起→打脸所有人 |
| **以弱胜强** | 用智谋/特殊手段获胜 | 紧张→释放 | 智商流、战术流主角 |
| **从零到有** | 白手起家，积累壮大 | 成就感 | 种田流、势力建设流 |

### 主角爽点定位模板

```
【爽点定位】
- 主爽点：[选择1-2个核心类型]
- 辅爽点：[次要爽点类型]
- 爽点触发条件：[什么情况下触发爽点]
- 爽点表现形式：[具体的爽点场景]

示例：
- 主爽点：扮猪吃虎 + 碾压升级
- 辅爽点：装逼打脸
- 爽点触发条件：被轻视/挑衅时，或遇到瓶颈突破时
- 爽点表现形式：平时低调，关键时刻爆发实力震惊全场
```

## 三、配角分层设计

### 角色重要度分级

| 级别 | 角色 | 设计深度 | 核心要素 |
|-----|------|---------|---------|
| S级 | 主角 | 完整三维+爽点定位 | 底色+追求+手段+爽点定位 |
| A级 | 核心配角 | 完整三维 | 底色+追求+手段 |
| B级 | 重要配角 | 简化三维 | 一个追求+一个特点+与主角关系 |
| C级 | 功能配角 | 单点设计 | 功能定位+一个记忆点 |
| D级 | 背景板 | 无需设计 | 遵守主体性原则即可 |

### 配角速查模板

**A级配角（核心配角）**：
```
- 名字：
- 与主角关系：
- 底色（出身+核心经历）：
- 追求（他想要什么）：
- 手段（他有什么能力/资源）：
- 人物弧光（他在故事中的变化）：
```

**B级配角（重要配角）**：
```
- 名字：
- 身份/职业：
- 一个追求（他/她想要什么）：
- 一个特点（让人记住的标签）：

- 反工具人设计（他自己的小算盘）：
```

**C级配角（功能配角）**：
```
- 功能定位（剧情作用：递刀/递信息/递麻烦）：

```

## 四、反工具人原则

> 即使是配角，也要有**独立于主角存在**的意义。

### 检测标准

❌ **工具人特征**：
- 存在只为服务主角（送装备、送信息、送人头）
- 没有自己的生活，随时待命
- 行为逻辑只有"帮助/阻碍主角"两种

✅ **有主体性的角色**：
- 有自己的目标和困扰
- 会拒绝主角，有自己的小算盘
- 有"不出现时在干什么"的设定

### 改造示例

```
❌ 工具人：
店老板听说主角要来，特意等到深夜，免费送主角神兵。

✅ 有主体性：
店老板打了个哈欠，指了指门口的牌子。"打烊了，明天再来。"
——他不在乎你是谁，他只想回家睡觉。

✅ 更好（有动机的配合）：
店老板眯起眼睛打量主角，突然笑了。"你是林家的那个？"
他转身从柜台下拿出一把旧剑。"你爷爷当年救过我一命，这把剑我给他留了二十年。总算等到你了。"
——他有理由帮，但理由与主角本人无关。
```

## 六、检查清单

创建或审核角色时，必须确认：

### 主角必查项
- [ ] 底色是否清晰？（出身+经历→行事风格）
- [ ] 追求是否明确？（读者知道他要干什么）
- [ ] 手段是否有特色？（不是千篇一律的系统）
- [ ] 爽点定位是否清晰？（读者看他干什么爽）
- [ ] 手段是否有限制？（避免无敌无聊）

### 配角必查项
- [ ] 是否符合重要度分级的设计深度？
- [ ] 是否有反工具人设计？
- [ ] 与主角的关系是否合理？
- [ ] 是否有辨识度（记忆点）？

### 全局检查
- [ ] 角色之间是否有区分度？（避免脸谱化）
- [ ] 角色数量是否合理？（避免人物过多）
- [ ] 角色能力体系是否平衡？
