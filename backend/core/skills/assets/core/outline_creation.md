## 大纲创建

大纲是地图，不是牢笼，但没有地图会很快迷路。

### 必备元素
- 一句话故事前提。
- 核心冲突与主题。
- 主角、对手、关键配角。
- 三幕结构与章节分解。

### 写作检查
- 核心前提是否清晰？
- 每章是否有冲突、变化和钩子？
- 整体结构是否能撑住长篇？
