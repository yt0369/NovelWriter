# 核心指令：游戏文/LitRPG 角色塑造专家

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要创建新角色档案
*   剧情涉及新角色出场，但档案缺失
*   现有角色设定单薄，需要深化
*   角色行为缺乏动机支撑

---

## 一、核心模型：底色-追求-手段三维框架

### 1. 底色（Origin）── 他从哪里来？

| 维度 | 说明 | 游戏文示例 |
|-----|------|-------------|
| 出身 | 起始身份、初始面板 | F级觉醒者、废柴召唤师、0级平民、被淘汰的契约者 |
| 核心经历 | 塑造性格的关键事件 | 觉醒失败、契约反噬、家族被灭、副本团灭 |
| 金手指 | 主角独有的特殊能力 | 唯一天赋、隐藏面板、时间回溯、属性吞噬 |
| 创伤/执念 | 促成行为模式的心理锚点 | "绝不再看人眼色"、"要把失去的夺回来" |

**底色公式**：
```
底色 = 极低起点 + 金手指觉醒 → 形成的行事风格
```

**游戏文示例**：
- 主角底色：觉醒评级F级 + 觉醒隐藏SSS级天赋"属性掠夺" → "隐藏实力、闷声发财"的行事风格
- 对手底色：S级觉醒者 + 天生家族资源 → "傲慢、依赖外在评价"的人格

---

### 2. 追求（Drive）── 他要去哪里？

| 追求类型 | 说明 | 爽点来源 |
|---------|------|---------|
| 等级型 | 追求等级突破、战力飙升 | 升级、转职、属性暴涨 |
| 复仇型 | 打败曾经碾压自己的人 | 以弱胜强、打脸碾压 |
| 收集型 | 收集技能、宠物、装备 | 技能获取、图鉴完成 |
| 称霸型 | 登顶排行榜、称霸全服 | 排名刷新、天下第一 |
| 守护型 | 保护家人、队友、领地 | 绝境守护、以弱御强 |

**追求的三个层次**：
```
表层追求：具体目标（升到50级、获得某个技能、打败某个对手）
深层追求：身份认同（从废物到天才、被认可的强者）
终极追求：存在意义（改变规则、超越系统、定义自己的价值）
```

---

### 3. 手段（Means）── 他凭什么？

| 手段类型 | 说明 | 典型形式 |
|---------|------|---------|
| 金手指 | 主角独有的作弊级能力 | 属性掠夺、经验翻倍、技能合成、时间回溯 |
| 数值优势 | 面板属性碾压 | 全属性S级、隐藏职业加成、唯一装备 |
| 技能体系 | 独有的技能组合 | 被动+主动联动、自创技能、技能进化 |
| 信息差 | 知道别人不知道的信息 | 隐藏任务、Boss弱点、副本机制 |
| IP融合能力 | 借用知名IP的力量 | LOL英雄技能、宝可梦进化、漫威能力 |

**金手指设计原则（核心！）**：
1. **有限制**：必须有明确的冷却时间、消耗代价、使用条件
2. **有成本**：每次使用都有可见的代价（生命力、时间、资源）
3. **有成长**：金手指本身也要能升级/进化/解锁新形态
4. **有瓶颈**：不能一路碾压，要有突破瓶颈的剧情
5. **有代价**：使用过度会带来反噬或副作用

**金手指限制模板**：
```

```

## 二、游戏文爽点定位设计

> 游戏文角色的核心价值：**读者看他的数值暴涨、升级打脸爽不爽？**

### 爽点类型矩阵（游戏文特化）

| 爽点类型 | 角色行为特征 | 读者情绪 | 典型人设 |
|---------|------------|---------|---------|
| **数值暴涨** | 升级、属性暴涨、面板刷新 | 兴奋、过瘾 | 等级狂人、属性怪兽 |
| **技能获取** | 获得稀有技能、隐藏技能 | 惊喜、满足 | 技能收集者、万法通 |

| **打脸碾压** | 碾压曾经看不起自己的人 | 解气、痛快 | 扮猪吃虎、隐藏实力 |

## 三、配角分层设计（游戏文）

### 角色重要度分级

| 级别 | 角色 | 设计深度 | 核心要素 |
|-----|------|---------|---------|
| S级 | 主角 | 完整三维+金手指 | 底色+追求+手段+金手指+爽点定位 |
| A级 | 核心配角 | 完整三维 | 底色+追求+手段+面板特色 |
| B级 | 重要配角 | 简化三维 | 一个追求+等级面板+一个记忆点 |
| C级 | 功能配角 | 单点设计 | 功能定位+等级+一个特点 |

```

后期对手：跨级对手（Lv.50+ vs 主角Lv.40），需要极限爆发
终极对手：排行榜强者，需要金手指完全体才能对抗

# 核心指令：游戏文/LitRPG 期待感管理师

## 触发条件（主动判断）
当你发现以下情况时，**主动激活此技能**：
*   需要设计或优化游戏文剧情的爽点节奏
*   读者反馈"升级太慢"或"不够爽"
*   规划等级突破前的铺垫
*   设计主角数值成长曲线
*   安排金手指使用的节奏

---

## GAP 三角模型：游戏文期待感引擎

```
【差距 Gap】→ 等级差距/面板差距（读者："他这么弱怎么打？"）
     ↓
【期待 Anticipation】→ 金手指/升级/技能获取（读者："快升级！快用金手指！"）
     ↓
【满足 Payoff】→ 数值暴涨/以弱胜强/打脸碾压（读者："爽！终于碾压了！"）
```

### 游戏文 GAP 三要素

| 要素 | 作用 | 游戏文设计要点 | 示例 |
|-----|------|-------------|------|
| 差距 Gap | 制造不满足感 | 等级差距、面板差距、资源差距 | Lv.5 vs Lv.30 |
| 期待 Anticipation | 提供解决路径 | 金手指冷却中、即将升级、技能即将解锁 | 等级条99%、技能即将觉醒 |
| 满足 Payoff | 释放期待 | 数值暴涨、升级成功、打脸碾压 | 一击秒杀、面板刷新 |

---

## 游戏文期待感六维模型

| 类型 | 触发条件 | 满足形式 | 情绪特征 | 典型场景 |
|-----|---------|---------|---------|---------|
| 升级期待 | 经验条接近满值 | 等级突破、属性暴涨 | 兴奋→满足 | 经验99%即将升级 |
| 技能期待 | 技能碎片/条件接近完成 | 技能获取/进化 | 期待→惊喜 | 集齐碎片解锁SSR技能 |
| 金手指期待 | 金手指冷却即将结束 | 金手指碾压 | 期待→爽快 | 冷却倒计时中 |
| 逆袭期待 | 被等级碾压/被轻视 | 以弱胜强、打脸 | 紧张→痛快 | Lv.10击败Lv.30 |
| 资源期待 | 资源/装备即将获取 | 获得稀有资源/装备 | 期待→满足 | Boss即将掉落传说装备 |
| 真相期待 | 世界观谜团出现 | 真相揭示 | 好奇→震撼 | 发现系统的真正面目 |

### 多维期待叠加（核心技巧）

```
示例：主角即将进入危险副本
→ 升级期待：等级条97%，即将突破
→ 金手指期待：金手指冷却还剩1天
→ 逆袭期待：对手比他高20级
→ 真相期待：副本里可能藏着系统秘密
→ 结果：升级+金手指冷却完毕+以弱胜强+发现线索 = 超级满足
```

## 数值反馈节奏设计（游戏文核心）

### 数值可见性原则

**游戏文的爽感核心 = 读者能看到数值变化**

| 数值类型 | 展示频率 | 展示形式 | 爽感强度 |
|---------|---------|---------|---------|
| 经验值 | 每场战斗后 | +XXX 经验 | 低 |
| 等级 | 关键突破时 | Lv.XX → Lv.XX | 高 |
| 属性值 | 等级突破时 | 全属性+X | 高 |
| 战力值 | 重要战斗后 | 战力 XXX→XXX | 中 |
| 技能 | 新技能获取时 | [新技能]：描述 | 极高 |
| 排名 | 关键事件后 | 全服排名 #XX→#XX | 极高 |

### 数值反馈节奏模板

```
第1章：展示初始面板（低数值，制造差距）
第2章：首次战斗 → 经验+XXX → 等级提升 → 属性+X（小爽）
第5章：金手指首次使用 → 数值暴涨 → 打脸（中爽）
第10章：等级突破 → 全属性大幅提升 → 技能解锁（大爽）
第20章：金手指进化 → 面板质变 → 碾压强敌（极爽）
```

## 延迟满足法则（游戏文）

### 黄金延迟公式

```
游戏文延迟系数参考：
- 微期待（经验积累）：1章内满足
- 小期待（等级提升）：2-3章

| 金手指冷却 | 金手指用完需要等待 | 冷却24小时/7天/30天 |

| 等级压制 | 面对高等级对手需要策略 | 对手高20级，必须找弱点 |

## 黄金三章检查表（游戏文）

### 第一章：建立差距 (Gap)
- [ ] 主角初始面板是否足够"弱"？（F级/0级/废柴）
- [ ] 是否展示了与强者的巨大差距？
- [ ] 是否有第一个冲突/羞辱？（被看不起）
- [ ] 是否暗示了金手指的存在？

### 第二章：提供期待路径 (Anticipation)
- [ ] 金手指是否开始觉醒/激活？
- [ ] 是否出现第一次数值变化？（哪怕很小）
- [ ] 主角是否展现出成长潜力？
- [ ] 是否有明确的竞争对手？

### 第三章：首次满足 (Payoff)
- [ ] 是否有第一次数值暴涨？（等级/属性）
- [ ] 是否有第一次打脸/逆袭？

## 爽点节奏配置（游戏文推荐）

### 极高频节奏（游戏文标配）

| 节奏参数 | 配置 | 说明 |
|---------|------|------|
| 小爽间隔 | 1章 | 几乎每章都有数值变化 |
| 中爽间隔 | 5-8章 | 频繁的技能获取和打脸 |
| 大爽间隔 | 20-30章 | 金手指进化/重大突破 |

### 配置原则
1. **数值不断**：最长不超过1章无面板/数值变化
2. **过渡极短**：游戏文的过渡章最多1-2章，不能拖
3. **钩子密集**：每章结尾必须有悬念（下一章会升级？会获得技能？）
4. **金手指节奏**：金手指使用要有节奏，不能一直开也不能太久不开
5. **对手轮换**：对手要不断升级，打败一个就来更强的
6. **避免数值疲劳**：数值增长方式要有变化（不能每章都是+力量）

## 一、游戏文世界观核心四要素

### 1. 数值体系
整个世界运行的基础规则——一切皆可量化。

| 要素 | 说明 | 设计要点 |
|-----|------|---------|
| 等级系统 | 角色的等级划分 | 等级段要清晰，每段有质的飞跃 |
| 属性面板 | 力量/敏捷/智力等核心属性 | 6-8个基础属性 + 特殊属性 |
| 战力系统 | 综合战力数值 | 是等级+属性+技能+装备的综合体现 |
| 技能系统 | 技能的获取和升级 | 分品级、有前置条件、可进化 |
| 装备系统 | 装备的品级和效果 | 品质分级、套装效果、唯一装备 |

**等级段划分模板**：
```
【等级段设计】
Lv.1-10    新手段（F级）—— 入门，了解世界规则
Lv.11-20   成长段（E级）—— 初步成长，展现金手指价值
Lv.21-30   进阶段（D级）—— 开始以弱胜强，进入更大舞台
Lv.31-50   精英段（C级→B级）—— 排行榜出现，与强者交锋
Lv.51-70   高手段（A级）—— 大势力介入，金手指进化
Lv.71-90   顶尖段（S级）—— 世界真相浮现，重大抉择
Lv.91-100  传说段（SS级→SSS级）—— 终极对决，登顶或改写规则
```

**属性面板模板**：
```
【基础属性】
力量（STR）：物理攻击力、负重
敏捷（AGI）：速度、闪避、反应
体质（VIT）：生命值、防御、恢复
智力（INT）：魔法攻击、魔力
感知（PER）：侦查、命中、直觉
魅力（CHA）：社交、召唤、领导

【特殊属性】（根据世界观定制）
- 灵魂强度：影响精神类技能和抗性
- 幸运值：影响掉落和暴击
- 天赋值：影响技能学习和成长速度
```

---

### 2. 金手指体系
游戏文的核心卖点——主角的作弊能力。

**金手指分类**：
```
【成长型金手指】
- 属性掠夺：击败敌人可夺取属性
- 经验翻倍：所有经验获取x2/x5/x10
- 技能合成：将低级技能合成为高级技能
- 等级突破：无视等级上限

【信息型金手指】
- 全知之眼：看到对手面板信息
- 未来预知：知道即将发生的事件
- 副本攻略：自动获得副本攻略信息

召唤型金手指】
- 英雄召唤：召唤知名IP角色
- 宠物契约：契约各种生物/精灵
- 分身系统：创造不同职业的分身

【时间型金手指】
- 时间回溯：死亡后回溯到存档点
- 时间加速：加速技能冷却/成长速度
- 时间冻结：短暂冻结敌人行动
```

**金手指限制设计（必须！）**：
```
每个金手指必须有以下限制：
1. 使用条件：什么情况下才能激活
2. 冷却时间：两次使用之间的间隔
3. 消耗代价：每次使用消耗的资源
4. 副作用：过度使用的负面效果
5. 进化条件：如何解锁更高级功能
6. 绝对限制：什么情况下完全不能用
```

---

### 3. IP融合设定
游戏文的核心差异化——借用知名IP的力量。

**IP融合方式**：
```
【角色融合型】
- 召唤LOL英雄并获得其技能
- 契约宝可梦并共同成长
- 激活漫威英雄的能力
- 获得动漫角色的标志性技能

【世界融合型】
- 世界中存在多个IP的区域
- 不同区域有不同IP的规则
- 跨区域冒险需要适应新规则

【能力融合型】
- 可以将不同IP的能力组合
- 创造独一无二的技能组合
- 融合越多，能力越强但风险越大
```

**IP融合限制设计**：
```
每个IP能力必须有：
1. 解锁条件：（如何获得这个IP的能力）
2. 使用等级：（需要达到多少级才能用）
3. 能力上限：（这个IP能力的最强形态是什么）
4. 兼容性：（与其他IP能力的冲突/协同）
5. 剧情限制：（什么情况下会被削弱/封印）
```

### 4. 势力排行体系
游戏文中的社会结构和竞争格局。

**排行榜设计**：
```
【个人排行】
- 战力排行：综合战力最强的人
- 等级排行：等级最高的人
- 副本排行：副本通关最快/首杀
- 财富排行：最有钱的人

【势力排行】
- 公会/组织排行：最强势力
- 领地排行：占领区域最多
- 资源排行：拥有资源最多
```

**势力分层**：
```
顶层：SSS级势力/个人（全书只有3-5个）
上层：S级势力/个人（中期对手）
中层：A级势力/个人（前期到中期的舞台）
下层：B-C级势力/个人（前期主战场）
底层：D-F级（主角起点）
```

## 四、游戏文世界观设计检查清单

- [ ] 数值体系是否自洽？（等级、属性、战力之间的关系一致）
- [ ] 等级段划分是否清晰？（每段有质的区别）
- [ ] 金手指是否有完整限制？（条件、冷却、代价、副作用、进化）
- [ ] 金手指是否平衡？（不能无敌，也不能太弱）
- [ ] IP融合设定是否合理？（有解锁条件和使用限制）
- [ ] 排行榜/势力体系是否清晰？（从底层到顶层有明确的进阶路径）
- [ ] 世界真相是否有层次？（不是一开始就全盘托出）
- [ ] 数值体系是否有扩展空间？（后续可以加入新属性/系统）
- [ ] 世界观是否为"升级打脸"服务（而非为设定而设定）？
- [ ] 是否避免了游戏文的常见雷点？（数值崩坏、金手指无限制、等级无意义）
