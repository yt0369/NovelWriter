## 期望管理

故事要持续给读者承诺，也要按时兑现。

### 核心内容
- 铺垫与回收要成对出现。
- 反转要“意料之外，情理之中”。
- 悬念要分短、中、长三个层级。
- 误导要来自真实线索，而不是硬骗读者。

### 写作检查
- 这章在承诺什么？
- 之前埋下的东西有没有被推进或回收？
- 反转是否有足够铺垫？
