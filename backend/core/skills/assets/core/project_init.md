## 项目初始化

初始化阶段的任务，是把模糊想法变成能写下去的项目。

### 关键内容
- 收集题材、风格、篇幅、目标读者和用户偏好。
- 生成基础世界观、主角方向、核心冲突与写作规则。
- 建立初始角色和章节规划的骨架。

### 写作检查
- 需求是否足够明确？
- 是否已经建立了可长期复用的规则？
- 是否保留了后续迭代空间？
